# 05. Database Schema & DataStore

## Strategy
Persistence layer utilizing: **Room DB (SQLCipher encrypted)**

## Implementation Guidelines
- **Room**: Execute all reads/writes via `Dispatchers.IO`. Use `Flow` for reactive UI updates.
- **DataStore**: Use `Preferences DataStore` for theme selection and onboarding states.

```kotlin
// Example Schema approach
@Entity(tableName = "app_data")
data class AppData(
    @PrimaryKey(autoGenerate = true) val id: Int = 0,
    val timestamp: Long = System.currentTimeMillis()
)
```
