# 02. UI/UX Design System — Water Log & Remind
```
This design system establishes a refreshing, fluid, and highly engaging interface that motivates users to drink water through satisfying visual feedback.
```
---
```
## 1. Color Palette (Material 3 Dynamic Mappings)
```

*   **Primary (Brand):** `#0288D1` (Clear Ocean Blue — core hydration color)
*   **On Primary:** `#FFFFFF` (Crisp White)
*   **Primary Container:** `#B3E5FC` (Soft Ripple Blue)

```
*   **Primary Font Family:** `Nunito` or `Quicksand` (Rounded, friendly, approachable sans-serif fonts that feel organic and soft).
*   **Alternative/Fallback:** `System Default Sans-Serif Rounded`.
```

## 3. Component Design Tokens
```
### 3.1 The "Liquid Fill" Dashboard Indicator
*   **Shape**: A large circle in the center of the dashboard.
*   **Animation**: Uses a sine-wave path animation in Jetpack Compose to simulate water rippling and rising based on the current progress percentage.
*   **Overlay**: The progress percentage text is overlaid on the circle, contrasting dynamically against the water fill level.
```
### 3.2 Quick-Add Preset Buttons
*   **Corner Radius**: Fully rounded (`CircleShape` or 50% radius).
*   **Iconography**: Simple vector icons inside a circular button (Glass, Bottle, Mug, Cup).
*   **Layout**: Horizontal scrollable row below the main progress indicator.
```
### 3.3 History Cards (Timeline)
*   **Corner Radius**: 12dp
*   **Elevation**: 0dp (flat design with subtle border)
*   **Layout**: Vertical timeline showing the time, beverage icon, and amount added.
*   **Swipe Action**: Swipe left on a history item to reveal a red "Delete/Undo" action.
```
---
```
## 4. Gesture Animations & Micro-Interactions
```
### 4.1 Liquid Drop Effect
*   **Interaction**: When the user taps a Quick-Add button, a small animated "water droplet" vector falls from the top of the screen and splashes into the main progress circle.
*   **Haptics**: Trigger a light haptic tap (`HapticFeedbackType.TextHandleMove`) upon pressing the button, and a distinct "pop" (`HapticFeedbackType.LongPress`) when the drop hits the circle.
```
### 4.2 Goal Completion Celebration
*   **Interaction**: When the progress hits 100% or more, trigger a confetti animation (using a Lottie file or Compose particle system) originating from the progress circle, accompanied by a bright visual flash of the Tertiary color (Gold).
```
---
```
## 5. Adaptive Layout Rules
*   **Compact (< 600dp):** Standard vertical stack. Progress circle at top, quick-add buttons below, recent history at the bottom. Bottom navigation bar.
*   **Medium/Expanded (> 600dp):** Two-pane layout. Left pane: Massive progress circle and quick add buttons. Right Pane: Scrollable daily timeline and weekly chart insights.
```
