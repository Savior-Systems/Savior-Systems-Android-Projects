# 12. Logging & Analytics

## Crashlytics (Firebase)
- Enabled only if user consents via UMP (or opt-in).
- Tracks fatal Android vitals.

## Analytics (Firebase)
- Custom Events:
  - `action_completed`
  - `ad_interstitial_shown`
  - `iap_purchased`
- **Privacy Constraint**: No PII (Personally Identifiable Information) or user-generated content is ever logged.
