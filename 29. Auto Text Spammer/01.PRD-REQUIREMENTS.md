# 01. Product Requirements Document (PRD)

## Project Context
- **App Name**: Auto Text Spammer Loop
- **Category**: Tools
- **Core Concept**: Repeat text blocks thousands of times for messaging limits testing or spam jokes.

## Problem Statement
Users need a highly focused, offline-first tool to achieve their goals without aggressive subscriptions, forced logins, or internet dependencies.

## Target Audience
- Users searching for "text repeater, spam templates, copy paste loop"
- Privacy-conscious individuals preferring offline functionality.

## Core Features
1. Immediate primary action on launch.
2. Highly polished Material You interface.
3. 100% offline data storage (Room DB (Saved spam templates)).

## Out of Scope
- Cloud syncing, User accounts, Complex social sharing APIs.
