# 02. UI/UX Design System
```
This document details visual guidelines and colors for **English-Bangla Vocab**.
```

```
## 1. Color Palette
The primary style uses **Royal Indigo** (`#3F51B5`) to deliver an academic, trustworthy, and high-focus feel.
```
### Dark Theme (Default)
*   **Background**: `#121212` (Pure Dark) / `#1C1E24` (Deep Indigo Slate)
*   **Primary (Brand)**: `#3F51B5` (Royal Indigo)
*   **Secondary (Brand Accent)**: `#7986CB` (Soft Blue/Lavender)
*   **On-Background (Text Primary)**: `#ECEFF1` (Off-white)
*   **On-Background (Text Secondary)**: `#9FA8DA` (Lavender Gray)
```
### Light Theme
*   **Background**: `#FAFAFA`
*   **Primary (Brand)**: `#3F51B5`
*   **Secondary (Brand Accent)**: `#5C6BC0`
*   **On-Background (Text Primary)**: `#1A237E` (Dark Blue)
*   **On-Background (Text Secondary)**: `#7986CB`
```
---
```
## 2. Typography
*   **Display Title**: `Outfit Bold`, 26sp
*   **Word Header**: `Outfit Medium`, 22sp
*   **Definition & Translations**: `Inter Regular`, 16sp
*   **IPA / Pronunciation**: `Inter Regular`, 14sp (Italics, e.g. */vəʊˈkæbjʊləri/*)
```
---
```
## 3. Flashcards Gestures
*   **Card Flip**: Tap flashcard to trigger a 3D rotation animation revealing the translation.
*   **Swipe Right**: Mark word as "Know" (moves to completed).
*   **Swipe Left**: Mark word as "Review" (leaves in active deck).
```
