# 12. Logging & Analytics
```
This document details logging rules and analytics setup for **English-Bangla Vocab**.
```
---
```
## 1. Privacy Mandate
*   **Zero PII**: Searched queries, bookmarked words, and translation definitions are strictly kept offline. We never log user definitions.
```
---
```
## 2. Analytics Events
```
### Event: Quiz Completed (`quiz_completed`)
Fired when a user finishes a vocabulary quiz session.
*   `quiz_score`: `INTEGER` (Number of correct answers, capped at 10)
*   `quiz_time_seconds`: `INTEGER` (Time elapsed to complete)
```
