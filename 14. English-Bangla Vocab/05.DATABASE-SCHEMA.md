## 1. Tables
```
### `words` (Pre-Populated)
Stores dictionary word maps.
*   `id`: `INTEGER` (Primary Key)
*   `english`: `TEXT` (Indexed for fast search lookup)
*   `bangla`: `TEXT` (Translations)
*   `ipa`: `TEXT` (IPA phonetic pronunciation)
*   `synonyms`: `TEXT` (Synonyms comma list)
*   `is_bookmarked`: `INTEGER` (Boolean flag)
```
### `study_records` (User Local Activity)
Stores study and flashcard history.
*   `id`: `INTEGER` (Primary Key)
*   `word_id`: `INTEGER` (Foreign Key -> `words(id)`)
*   `is_memorized`: `INTEGER` (Boolean flag)
*   `last_reviewed`: `INTEGER` (Timestamp)
```

## 2. Room Entities Code
```
```kotlin
package com.saviorsystems.englishbanglavocab.data.database.entities
```
import androidx.room.Entity
import androidx.room.Index
import androidx.room.PrimaryKey
```
@Entity(
    tableName = "words",
    indices = [Index(value = ["english"])]
)
data class WordEntity(
    @PrimaryKey(autoGenerate = true) val id: Long = 0L,
    val english: String,
    val bangla: String,
    val ipa: String?,
    val synonyms: String?,
    val isBookmarked: Boolean = false
)

## 3. Query Interfaces (VocabDao)
```
```kotlin
package com.saviorsystems.englishbanglavocab.data.database
```
import androidx.room.*
import com.saviorsystems.englishbanglavocab.data.database.entities.WordEntity
import kotlinx.coroutines.flow.Flow
```
@Dao
interface VocabDao {
    @Query("SELECT * FROM words WHERE english LIKE :query || '%' ORDER BY english ASC LIMIT 50")
    fun searchWords(query: String): Flow<List<WordEntity>>
```
    @Query("SELECT * FROM words WHERE isBookmarked = 1")
    fun getBookmarkedWords(): Flow<List<WordEntity>>
```
    @Update
    suspend fun updateWord(word: WordEntity)
}
