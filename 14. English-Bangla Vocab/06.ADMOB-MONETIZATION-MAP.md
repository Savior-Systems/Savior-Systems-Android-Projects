# 06. AdMob Monetization Map
`
This document outlines monetization layout and ad placements for **English-Bangla Vocab**.
`
---
`
## 1. Placements
*   **Adaptive Banner**: Docked at the bottom of the Dictionary Search layout.
*   **Interstitial Ad**: Triggered upon completion of a 10-word quiz session.
*   **Zero-Ad Focus Zones**: Flashcards swipe deck, bookmark lists, and spelling/definition screens are kept 100% ad-free.
`
---
`
## 2. Cooldown Implementation
To protect user experience, quizzes must check the `180-second` global cooldown before displaying any interstitial ad.
`
```kotlin
package com.saviorsystems.englishbanglavocab.utils
`
object AdMobHelper {
    private var lastAdShowTime: Long = 0L
    private const val COOLDOWN_MS = 180_000L
`
    fun canShowInterstitial(): Boolean = (System.currentTimeMillis() - lastAdShowTime) >= COOLDOWN_MS
    fun markAdShown() { lastAdShowTime = System.currentTimeMillis() }
}
```
