# 03. Functional Flows
`
This document details user transition maps and flows for **Fuel Mileage Log**.
`
---
`
## 1. Quick Fuel Logging Flow
```mermaid
sequenceDiagram
    participant User
    participant Dashboard
    participant InputForm
    participant RoomDB
`
    User->>Dashboard: Tap "+" Floating Action Button
    Dashboard->>InputForm: Slide up input fields (Odometer, Volume, Cost)
    User->>InputForm: Input odometer=45210, volume=42.5, cost=55.0
    User->>InputForm: Click "Save Log"
    InputForm->>RoomDB: Insert FuelRecordEntity
    RoomDB-->>Dashboard: Refresh list Flow
    Dashboard->>User: Play feedback vibration & update mileage stats
```
`
---
`
## 2. Screen & Dialog Hierarchy
*   **Main Screen (Dashboard)**:
    *   Top: Average efficiency stat display (e.g. `6.8 L/100km` or `34.5 MPG`).
    *   Center: History list displaying past logs sorted by odometer reading.
    *   Bottom: Settings gear button and "+" Log Button.
*   **Settings Screen**:
    *   Unit systems selector (Metric vs Imperial).
    *   CSV Export/Import utility.
    *   Delete all log histories.
