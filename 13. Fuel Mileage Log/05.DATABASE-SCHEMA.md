# 05. Database Schema

This document details the SQLite/Room database schema for **Fuel Mileage Log**.

---

## 1. Table: `fuel_records`
Stores the user's sequential fuel purchases and odometer logs.

| Column Name | SQLite Data Type | Constraints | Description |
| :--- | :--- | :--- | :--- |
| `id` | `INTEGER` | PRIMARY KEY AUTOINCREMENT | Unique record ID |
| `odometer` | `REAL` | NOT NULL | Current vehicle mileage readout |
| `volume_liters` | `REAL` | NOT NULL | Amount of fuel filled |
| `total_cost` | `REAL` | NOT NULL | Total cost of purchase |
| `timestamp` | `INTEGER` | NOT NULL | Time of log creation |

---

## 2. Room Entity Kotlin Implementation

```kotlin
package com.saviorsystems.fuelmileagelog.data.database.entities

import androidx.room.ColumnInfo
import androidx.room.Entity
import androidx.room.Index
import androidx.room.PrimaryKey

@Entity(
    tableName = "fuel_records",
    indices = [Index(value = ["odometer"])]
)
data class FuelRecordEntity(
    @PrimaryKey(autoGenerate = true) val id: Long = 0L,
    val odometer: Double,
    @ColumnInfo(name = "volume_liters") val volumeLiters: Double,
    @ColumnInfo(name = "total_cost") val totalCost: Double,
    val timestamp: Long = System.currentTimeMillis()
)
```

## 3. Database DAO Interfaces

```kotlin
package com.saviorsystems.fuelmileagelog.data.database

import androidx.room.*
import com.saviorsystems.fuelmileagelog.data.database.entities.FuelRecordEntity
import kotlinx.coroutines.flow.Flow

@Dao
interface FuelDao {
    @Query("SELECT * FROM fuel_records ORDER BY odometer DESC")
    fun getAllRecords(): Flow<List<FuelRecordEntity>>

    @Insert(onConflict = OnConflictStrategy.REPLACE)
    suspend fun insertRecord(record: FuelRecordEntity): Long

    @Delete
    suspend fun deleteRecord(record: FuelRecordEntity)

    @Query("DELETE FROM fuel_records")
    suspend fun clearAllRecords()
}
```
