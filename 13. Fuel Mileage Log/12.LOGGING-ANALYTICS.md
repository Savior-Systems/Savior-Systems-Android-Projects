# 12. Logging & Analytics

This document details logging rules and analytics setup for **Fuel Mileage Log**.

---

## 1. Privacy Mandate
*   **Zero PII**: No locations, prices, database records, or vehicle descriptions are uploaded to telemetry.
*   **Opt-out**: Global analytics toggle exists in Settings.

---

## 2. Analytics Events

### Event: Fuel Record Logged (`fuel_record_logged`)
Fired when a user records a fill-up.
*   `unit_system`: `TEXT` ("metric" or "imperial")
*   `has_cost`: `INTEGER` (0 = No, 1 = Yes)

### Event: Analytics Settings Changed (`settings_toggled`)
Fired when user changes preference toggles.
