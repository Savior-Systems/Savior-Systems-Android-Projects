# 02. UI/UX Design System

## Global Theme Concept
A bespoke Material 3 configuration using a unique color palette designed specifically for Tip Calculator Split Bills.

## Typography
- **Heading**: 'Outfit' (Google Fonts)
- **Body**: 'Inter' or system default 'Roboto'

## Color Palette Strategy
- **Primary**: Distinctive vibrant accent (needs to be defined uniquely).
- **Surface**: Pure dark mode AMOLED black (#000000) or clean white (#FFFFFF) with high contrast.

## Core Screens
1. **Home Screen**: Direct access to primary function.
2. **Settings**: Minimalist options (Dark mode toggle, Pro upgrade ad).
3. **History/Vault (if applicable)**: List of saved items.
