# 02. UI/UX Design System
```
This document details the visual guidelines, themes, and interactive patterns for the **Minimalist To-Do** application.
```
---
```
## 1. Visual Philosophy
The design philosophy is **Functional Minimalism**. Every UI element must justify its presence. We replace heavy containers, lines, and boxes with generous white space, subtle contrast, and smooth gesture transitions.
```
---
```
## 2. Color Palette
The primary theme is built around **Charcoal Gray** (`#37474F`), offering a highly professional, focused, and executive aesthetic.
```
### Dark Theme (Default)
*   **Background**: `#121212` (Pure Dark) / `#1E272C` (Soft Slate surface)
*   **Primary (Brand)**: `#37474F` (Charcoal Gray)
*   **Secondary (Brand Accent)**: `#4DB6AC` (Soft Teal - Neon dark variant)
*   **On-Background (Text Primary)**: `#ECEFF1` (Off-white)
*   **On-Background (Text Secondary)**: `#90A4AE` (Muted Blue-Gray)
*   **Success (Complete Gesture)**: `#2E7D32` (Forest Green)
*   **Error (Delete Gesture)**: `#C62828` (Crimson Red)
```
### Light Theme
*   **Background**: `#FAFAFA` (Off-white)
*   **Primary (Brand)**: `#37474F` (Charcoal Gray)
*   **Secondary (Brand Accent)**: `#00897B` (Soft Teal)
*   **On-Background (Text Primary)**: `#263238` (Very Dark Gray)
*   **On-Background (Text Secondary)**: `#78909C` (Muted Slate)
*   **Success (Complete Gesture)**: `#4CAF50` (Vibrant Green)
*   **Error (Delete Gesture)**: `#EF5350` (Soft Red)
```

## 3. Typography
The app uses a modern sans-serif typeface hierarchy (e.g., `Outfit` or `Inter` from Google Fonts).
```
*   **Display Title**: `Outfit Bold`, 28sp (App Brand, Empty State)
*   **Header / Category List**: `Outfit Medium`, 16sp (Caps / Muted tag tabs)
*   **Task Title**: `Inter Medium`, 16sp (High readability for tasks)
*   **Task Metadata**: `Inter Regular`, 12sp (Due dates, priority indicators)
*   **Quick-Add Input**: `Inter Regular`, 15sp (Prompt text)
```
---
```
## 4. Components & Elevation
*   **Task Card**:
    *   No card border.
    *   Flat styling with slightly contrasting surface background (`#1E272C` on dark theme, `#FFFFFF` with slight elevation on light theme).
    *   Rounded corners: `12dp`.
*   **Category Tags**:
    *   Capsule shaped chips.
    *   Active: Solid Secondary Accent (`#00897B` / `#4DB6AC`) with White text.
    *   Inactive: Muted outline or light surface color with secondary text.
*   **Quick-Add Bar**:
    *   Anchored to the absolute bottom of the main activity, styled with Glassmorphic semi-transparency using Compose's background alpha.
```
---
```
## 5. Micro-Animations & Gestures
1.  **Swipe-To-Dismiss (Task Item)**:
    *   As the user swipes right, the card shifts to reveal a green background containing a Checkmark icon that grows from scale `0.5` to `1.1` as threshold is crossed.
    *   As the user swipes left, the card shifts to reveal a red background containing a Trash icon.
2.  **Category Tag Select**:
    *   Selected category has an animated width expansion (`animateDpAsState`) to show selection.
3.  **Task Completed Transition**:
    *   When checked off, the text transitions to cross out (strikethrough) and fades to `alpha = 0.4` using Compose animation transitions before sliding off the screen or moving to bottom of list.
