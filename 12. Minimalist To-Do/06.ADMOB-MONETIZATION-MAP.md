# 06. AdMob Monetization Map
`
This document outlines the high-yield, user-friendly monetization model for **Minimalist To-Do**, detailing the ad integrations and zero-ad focus zones.
`
---
`
## 1. Monetization Philosophy
To maintain the app's clean minimalist feel, ads are kept non-obtrusive. We implement **Zero-Ad Focus Zones** where the user performs critical productivity actions (task entry and checking tasks off) to guarantee zero friction.
`
---
`
## 2. Ad Formats & Placements
`
### A. Adaptive Banner Ad
*   **Location**: Anchored to the bottom of the main task list.
*   **Behavior**: Auto-adjusts size based on screen configuration. Suspended when the virtual keyboard is open (so task entry remains unobstructed).
*   **Unit ID (Android Test)**: `ca-app-pub-3940256099942544/6300978111`
`
### B. Interstitial Ad
*   **Trigger Events**:
    1.  Tapping "Clear All Completed Tasks" in settings.
    2.  Exiting the Settings screen back to the Main Task List.
*   **Guards**:
    *   **180s Cooldown Timer**: Checked globally before attempting to load or show an interstitial ad.
    *   **No-Show condition**: Never trigger when the user is actively typing a task.
*   **Unit ID (Android Test)**: `ca-app-pub-3940256099942544/1033173712`
`

## 4. Kotlin Integration Code (AdMobHelper)
`
```kotlin
package com.saviorsystems.minimalisttodo.utils
`
import android.content.Context
import com.google.android.gms.ads.AdRequest
import com.google.android.gms.ads.LoadAdError
import com.google.android.gms.ads.interstitial.InterstitialAd
import com.google.android.gms.ads.interstitial.InterstitialAdLoadCallback
`
object AdMobHelper {
    private var lastAdShowTime: Long = 0L
    private const val COOLDOWN_MS = 180_000L // 3 minutes in milliseconds
    private var mInterstitialAd: InterstitialAd? = null
`
    fun loadInterstitialAd(context: Context, adUnitId: String) {
        val adRequest = AdRequest.Builder().build()
        InterstitialAd.load(
            context,
            adUnitId,
            adRequest,
            object : InterstitialAdLoadCallback() {
                override fun onAdLoaded(interstitialAd: InterstitialAd) {
                    mInterstitialAd = interstitialAd
                }
`
                override fun onAdFailedToLoad(loadAdError: LoadAdError) {
                    mInterstitialAd = null
                }
            }
        )
    }
`
    fun showInterstitialAd(context: Context, adUnitId: String, onAdClosed: () -> Unit) {
        val currentTime = System.currentTimeMillis()
        val isCooldownPassed = (currentTime - lastAdShowTime) >= COOLDOWN_MS
`
        if (mInterstitialAd != null && isCooldownPassed) {
            mInterstitialAd?.fullScreenContentCallback = object : com.google.android.gms.ads.FullScreenContentCallback() {
                override fun onAdDismissedFullScreenContent() {
                    lastAdShowTime = System.currentTimeMillis()
                    mInterstitialAd = null
                    loadInterstitialAd(context, adUnitId) // Preload next
                    onAdClosed()
                }
`
                override fun onAdFailedToShowFullScreenContent(adError: com.google.android.gms.ads.AdError) {
                    mInterstitialAd = null
                    onAdClosed()
                }
            }
            mInterstitialAd?.show(context as android.app.Activity)
        } else {
            // Cooldown active or ad not loaded yet, skip ad and continue
            onAdClosed()
        }
    }
}
```
