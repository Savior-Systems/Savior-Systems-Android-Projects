# 01. Product Requirement Document (PRD)

## 1. Vision & Strategy
*   **App Name**: Minimalist To-Do
*   **Category**: Productivity
*   **Core Objective**: Build a highly performant, privacy-first, 100% offline-first simple task manager. It directly targets search intent for the keywords: `minimalist to do list`, `simple task manager`, `offline todo app`, `daily checklist`, and `private to-do list`.
*   **Value Proposition**: Zero friction, instantaneous load times, gesture-driven controls (swipe to complete/delete), and 100% local storage. Users can enter a task in under 5 seconds from app launch.

---

## 2. User Personas
### Persona A: Sarah (Busy Architecture Student)
*   **Needs**: Needs to capture quick, unstructured tasks (e.g., "Buy trace paper," "Print floor plans") between classes without getting bogged down by task categories, due dates, tags, and sync screens.
*   **Pain Points**: Standard apps require logging in, load slowly, show heavy ads on entry, and make task entry a multi-step wizard.
*   **How the App Solves it**: Sarah opens the app, types the task immediately into the bottom quick-add bar, and taps Send. The task is created instantly.

### Persona B: David (Minimalist Product Designer)
*   **Needs**: A clean, aesthetically pleasing, and distraction-free visual environment to keep trace of his 3-5 daily critical items.
*   **Pain Points**: Messy UIs filled with badges, notifications, and complex navigation menus that scatter focus.
*   **How the App Solves it**: The application uses a Charcoal Gray tone with high-contrast text and simple swipe gestures, leaving the display quiet and focused.

---

## 3. Core Feature List
1.  **Frictionless Fast Capture**:
    *   An anchored text entry field at the bottom of the list screen that is focused or ready immediately on tap.
    *   Pressing Enter or tapping the send icon saves the task instantly.
2.  **Gesture-Based Checklist Management**:
    *   **Swipe Right**: Marks task as completed (dynamic green checkmark background indicator).
    *   **Swipe Left**: Triggers delete/archive action (dynamic crimson trash background indicator).
3.  **Category Filtering & Organization**:
    *   Horizontal tag bar at the top of the main screen (e.g., "All", "Work", "Personal", "Shopping", plus a "+" button for custom categories).
4.  **Local Smart Sorting Options**:
    *   Sorting parameters: Creation Date (Default), Priority Level (High/Medium/Low), and Due Date (Optional).
5.  **App Configuration Settings**:
    *   Light / Dark Mode theme control.
    *   Clear All Completed Tasks action.
    *   Local backup / restore (JSON import/export).
    *   Task notification reminder settings.

---

## 4. zero-Ad Zones & Non-Intrusive Monetization
*   **Zero-Ad Zones**:
    *   The main task list interface is kept free of high-frequency interstitial ads.
    *   The Task Addition flow and Task Complete/Delete swipe gestures are strictly zero-ad to avoid interrupting user workflow.
*   **Monetization**:
    *   A small, clean banner ad anchored to the bottom/top of the screen.
    *   An interstitial ad triggers only when performing bulk operations (e.g., "Clear All Completed Tasks") or when navigating out of the Settings screen, guarded by a **180-second global cooldown timer**.

---

## 5. Key Metrics & KPIs
*   **Speed-to-Capture**: Action time from launch to task save must be < 5 seconds.
*   **App Launch Time**: Cold launch to interactive state under 1.5 seconds.
*   **User Retention**: High 30-day retention due to zero-friction utility value.
*   **Database Reliability**: 0% data corruption / sync issues because of robust Room SQLite local storage.
