# 12. Logging & Analytics
```
This document defines the privacy-first logging standards and non-PII Firebase Analytics configuration for **Minimalist To-Do**.
```
---
```
## 1. Privacy First Mandate
To maintain strict compliance with GDPR and Google Play's Personal Privacy regulations:
*   **Zero PII Collection**: Task titles, categories, notes, or descriptions must **never** be logged to analytics dashboards or text logs.
*   **Local Debug Logging**: In non-production builds (`BuildConfig.DEBUG`), logs are outputted to logcat. Production builds must strip all debug logging.
*   **User Opt-Out**: Provide a global toggle in settings to completely disable Firebase Analytics tracking.
```

## 2. Analytics Events & Parameters
```
### Event A: Task Created (`task_created`)
Fired when the user inserts a new task. Used to monitor retention and engagement.
*   `priority_level`: `INTEGER` (0 = Low, 1 = Medium, 2 = High)
*   `has_due_date`: `INTEGER` (0 = No, 1 = Yes)
```
### Event B: Task Completed (`task_completed`)
Fired when a task is checked off (swiped right).
*   `days_to_complete`: `INTEGER` (Days elapsed since creation date, capped at 99)
*   `priority_level`: `INTEGER`
```
### Event C: Bulk Operation (`bulk_operation_triggered`)
Fired when clean operations are executed.
*   `operation_type`: `TEXT` ("clear_completed", "category_reset")
```
### Event D: Category Added (`category_added`)
Fired when a custom category tag is created.
*   `color_hex_type`: `TEXT` (Color group category selected)
```

## 3. Analytics Integration Helper (Firebase / Local Analytics)
```
```kotlin
package com.saviorsystems.minimalisttodo.utils
```
import android.content.Context
import android.os.Bundle
import com.google.firebase.analytics.FirebaseAnalytics
```
object AnalyticsHelper {
    private var firebaseAnalytics: FirebaseAnalytics? = null
    private var isTrackingEnabled = true
```
    fun initialize(context: Context, enabled: Boolean) {
        isTrackingEnabled = enabled
        if (enabled) {
            firebaseAnalytics = FirebaseAnalytics.getInstance(context)
        }
    }
```
    fun setTrackingEnabled(enabled: Boolean) {
        isTrackingEnabled = enabled
        if (!enabled) {
            firebaseAnalytics = null
        }
    }
```
    fun logTaskCreated(priority: Int, hasDueDate: Boolean) {
        if (!isTrackingEnabled) return
        val bundle = Bundle().apply {
            putInt("priority_level", priority)
            putInt("has_due_date", if (hasDueDate) 1 else 0)
        }
        firebaseAnalytics?.logEvent("task_created", bundle)
    }
```
    fun logTaskCompleted(daysToComplete: Int, priority: Int) {
        if (!isTrackingEnabled) return
        val bundle = Bundle().apply {
            putInt("days_to_complete", daysToComplete)
            putInt("priority_level", priority)
        }
        firebaseAnalytics?.logEvent("task_completed", bundle)
    }
```
    fun logBulkOperation(operationType: String) {
        if (!isTrackingEnabled) return
        val bundle = Bundle().apply {
            putString("operation_type", operationType)
        }
        firebaseAnalytics?.logEvent("bulk_operation_triggered", bundle)
    }
}
