# 05. Database Schema

This document details database rules for **Device Info Specs**.

---

## 1. Storage Choice
Since **Device Info Specs** is a dynamic diagnostic dashboard that queries hardware metrics in real-time on launch, the application has **zero offline database requirements**.

No SQLite databases, Room tables, shared keys, or DataStore preferences are created or used. All data is managed in memory during the active session.
