## 1. Color Palette
The primary style uses **Modern Slate** (`#455A64`) to deliver an industrial, technical, and high-performance engineering diagnostic look.
```
### Dark Theme (Default)
*   **Background**: `#121212` / `#1B2224` (Deep Slate Charcoal)
*   **Primary (Brand)**: `#455A64` (Modern Slate)
*   **Secondary (Brand Accent)**: `#90A4AE` (Steel Blue)
*   **On-Background (Text Primary)**: `#ECEFF1` (Slate Ice White)
*   **On-Background (Text Secondary)**: `#CFD8DC` (Muted Blue-Gray)
```
### Light Theme
*   **Background**: `#FAFAFA`
*   **Primary (Brand)**: `#37474F`
*   **Secondary (Brand Accent)**: `#78909C`
*   **On-Background (Text Primary)**: `#21272A`
*   **On-Background (Text Secondary)**: `#546E7A`
```
---
```
## 2. Typography
*   **Display Title**: `Outfit Bold`, 26sp
*   **Hardware Readout Value**: `Outfit Medium`, 20sp (High contrast numbers for RAM/CPU stats)
*   **Technical Labels**: `Inter Regular`, 13sp
