# 09. Testing & Quality Assurance Plan

This document outlines testing strategies, unit test implementations, and manual QA checklists for **Device Info Specs**.

---

## 1. Automated Unit Test Cases (Parser Checks)

```kotlin
package com.saviorsystems.deviceinfospecs

import org.junit.Assert.assertTrue
import org.junit.Test

class CpuMetricsTest {

    @Test
    fun verifyCoreReportingParser() {
        val coresCount = Runtime.getRuntime().availableProcessors()
        
        // Cores reported must be greater than or equal to 1
        assertTrue(coresCount >= 1)
    }
}
```
