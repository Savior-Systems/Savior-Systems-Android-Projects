# 07. ASO & Play Store Listing

This document details Google Play store descriptions and search keywords for **Device Info Specs**.

---

## 1. Keywords
*   **Primary**: `system monitor`, `device hardware info`, `cpu specs`
*   **Secondary**: `ram stats dashboard`, `device testing tools`, `hardware diagnostic`

---

## 2. Listing Copy

### App Title (Max 30 chars)
`Device Info: System Specs`

### Short Description (Max 80 chars)
`Complete, offline device hardware specs & diagnostic testing dashboard. Monitor CPU.`

### Long Description (Max 4000 chars)
```
Verify your device's hardware, CPU, memory, and specs instantly with Device Info: System Specs. A private, 100% offline-first utility that reports system readouts.

No data leaves your device, and no internet access permissions are required. Get direct details of your processor, memory stats, camera details, and active sensors.

CORE FEATURES:
• System Dashboard: Real-time CPU frequencies, RAM storage bars, and battery levels.
• CPU Diagnostic Tabs: Complete processor specs (cores, architecture, GPU details).
• Hardware Specifications: Readouts for screen density, display sizes, cameras, and local build details.
• Sensor Inspector: Live testing grids for Gyroscope, Accelerometer, and Magnetometer readouts.
• 100% Offline: Safe hardware audits with zero data collection.

Great for validating device capabilities offline. Download today!
```
