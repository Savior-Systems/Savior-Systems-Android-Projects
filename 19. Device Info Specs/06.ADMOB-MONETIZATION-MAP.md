# 06. AdMob Monetization Map
`
This document outlines AdMob integrations for **Device Info Specs**.
`
---
`
## 1. Placements
*   **Adaptive Banner**: Docked at the bottom of the System Dashboard.
*   **Interstitial Ad**: Triggered when the user runs a hardware benchmark test or exits Settings.
*   **Zero-Ad Focus Zones**: Live CPU frequency meters and sensor graphs are kept 100% ad-free.
`
---
`
## 2. 180s Cooldown Guard
A global cooldown is enforced to prevent ad spam.
`
```kotlin
package com.saviorsystems.deviceinfospecs.utils
`
object AdMobHelper {
    private var lastAdShowTime: Long = 0L
    private const val COOLDOWN_MS = 180_000L
`
    fun canShowInterstitial(): Boolean = (System.currentTimeMillis() - lastAdShowTime) >= COOLDOWN_MS
    fun markAdShown() { lastAdShowTime = System.currentTimeMillis() }
}
```
