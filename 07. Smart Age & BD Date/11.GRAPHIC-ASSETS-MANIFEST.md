## 2. Adaptive Icon XML Layout
`
The launcher icon uses the following vector configuration to comply with Material 3 dynamic color-shifting and adaptive guidelines.
`
*   **Background Layer (`res/drawable/ic_launcher_background.xml`)**: Solid color hex `#AD1457` (Warm Rose) with a subtle concentric circle watermark.
*   **Foreground Layer (`res/drawable/ic_launcher_foreground.xml`)**: Vector layout showing a calendar grid overlapping a clock/countdown vector, highlighted in Rose Gold (`#FFD54F` or white).
*   **Monochrome Layer (`res/drawable/ic_launcher_monochrome.xml`)**: Vector representation that allows system-wide Material You theme color tinting.

## 3. Play Store Screenshot Structure
`
We construct 4 high-fidelity marketing frames targeting Google Play Console requirements.
`
1.  **Frame 1 (Main Dashboard)**: Showcase the clean, modern input fields for DOB and Target Date. Title: *Calculate Age Instantly*.
2.  **Frame 2 (Result Sheet)**: Detail the precise breakdowns (years, months, weeks, hours, seconds). Title: *Get Down to the Second Accuracy*.
3.  **Frame 3 (Birthday Countdowns)**: Display the local registry list showing family countdown cards. Title: *Track Upcoming Birthdays*.
4.  **Frame 4 (Date Arithmetic)**: Show adding or subtracting intervals from arbitrary starting dates. Title: *Calculate Date Differences Easily*.
