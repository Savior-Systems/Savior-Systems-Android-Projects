# 05. Database Schema & Room DAO — Smart Age & BD Date
```
All data resides completely locally on the device. We use Android's **Room** database library for profile management and logging.
```
---
```
## 1. Entity: `birthday_profiles`
```
This table stores saved family, friends, or historical dates for countdowns.
```
| Column Name | SQLite Data Type | Constraints / Keys | Description |
| :--- | :--- | :--- | :--- |
| `id` | INTEGER | PRIMARY KEY AUTOINCREMENT | Unique record ID |
| `name` | TEXT | NOT NULL | Name of the person |
| `birth_date` | TEXT | NOT NULL | ISO-8601 formatted date (`YYYY-MM-DD`) |
| `notes` | TEXT | NULLABLE | Custom text field |
| `avatar_color` | INTEGER | NOT NULL | HSL color index for UI avatar rendering |
| `alarm_scheduled` | INTEGER | DEFAULT 0 | 0 = No alarm, 1 = Alarm scheduled in system |
```
---
```
## 2. Entity: `calculation_history`
```
This table persists the search queries for auditing government-age requirements.
```
| Column Name | SQLite Data Type | Constraints / Keys | Description |
| :--- | :--- | :--- | :--- |
| `id` | INTEGER | PRIMARY KEY AUTOINCREMENT | Unique record ID |
| `dob` | TEXT | NOT NULL | Date of Birth (`YYYY-MM-DD`) |
| `target_date` | TEXT | NOT NULL | Comparison date (`YYYY-MM-DD`) |
| `calculated_at` | INTEGER | NOT NULL | Epoch millisecond timestamp of calc run |
```

```
## 3. Data Access Object (DAO) Interface
```
```kotlin
package com.saviorsystems.smartage.data.database
```
import androidx.room.*
import kotlinx.coroutines.flow.Flow
```
@Dao
interface BirthdayDao {
```
    @Query("SELECT * FROM birthday_profiles ORDER BY birth_date ASC")
    fun getAllProfilesFlow(): Flow<List<BirthdayProfileEntity>>
```
    @Query("SELECT * FROM birthday_profiles WHERE id = :id")
    suspend fun getProfileById(id: Long): BirthdayProfileEntity?
```
    @Insert(onConflict = OnConflictStrategy.REPLACE)
    suspend fun insertProfile(profile: BirthdayProfileEntity): Long
```
    @Delete
    suspend fun deleteProfile(profile: BirthdayProfileEntity)
```
    @Query("SELECT * FROM calculation_history ORDER BY calculated_at DESC LIMIT 50")
    fun getCalculationHistoryFlow(): Flow<List<CalculationHistoryEntity>>
```
    @Insert(onConflict = OnConflictStrategy.REPLACE)
    suspend fun insertHistoryEntry(entry: CalculationHistoryEntity)
```
    @Query("DELETE FROM calculation_history")
    suspend fun clearHistory()
}
