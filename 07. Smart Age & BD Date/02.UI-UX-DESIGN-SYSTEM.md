# 02. UI/UX Design System — Smart Age & BD Date
```
Each application in the Savior Systems portfolio maintains visual isolation. The design system for **Smart Age & BD Date** is built around a premium, welcoming "Warm Rose" palette.
```
---
```
## 1. Color System (Warm Rose)
```
### Light Theme
*   **Primary (`#AD1457`)**: Rich Rose Accent for primary action buttons, FABs, and selected states.
*   **Primary Container (`#FCE4EC`)**: Light pink highlight for cards and chips.
*   **Secondary (`#880E4F`)**: Deep Plum for active tabs and headers.
*   **Tertiary (`#00796B`)**: Emerald Green for positive date intervals and success highlights.
*   **Background (`#FFFBFB`)**: Soft pinkish off-white background to prevent stark white glare.
*   **Surface (`#FFFFFF`)**: Clean white for lists and calculator form cards.
*   **On-Background (`#1A0C11`)**: Deep charcoal with a hint of rose for maximum readability.
```
### Dark Theme
*   **Primary (`#F06292`)**: Soft rose pink for high-contrast highlights.
*   **Primary Container (`#880E4F`)**: Deep plum container backing.
*   **Secondary (`#F48FB1`)**: Soft pastel pink.
*   **Tertiary (`#4DB6AC`)**: Teal accent for calculations.
*   **Background (`#12080B`)**: Deepest plum-black.
*   **Surface (`#1C0E13`)**: Soft dark card background.
*   **On-Background (`#ECE0E2`)**: Crisp warm-grey text.
```

```
## 2. Typography Standard
*   **Font Family**: **Outfit** (via Google Fonts Integration).
*   **Display Large (Age Value)**: 56sp, Bold, tracking -0.5px. Used for displaying the final age years.
*   **Title Large (Card Header)**: 20sp, Semi-Bold, tracking 0.15px.
*   **Body Medium (Data labels)**: 14sp, Regular, tracking 0.25px.
```
---
```
## 3. Micro-Animations
```
### Animation A: Countdown Clock Pulse
*   **Trigger**: Active birthday countdown view.
*   **Behavior**: The "seconds" field performs a subtle scale pulse (`scaleY` & `scaleX` from 1.0 to 1.15 to 1.0) every 1000ms using a `spring()` spec with `StiffnessLow`.
```
### Animation B: Card Slide & Reveal
*   **Trigger**: Submitting a date calculation.
*   **Behavior**: The result sheet slides up from the bottom boundary with a custom cubic bezier (`0.34, 1.56, 0.64, 1`) over 350ms, giving a premium tactile bounce.
```
