# 01. Product Requirement Document (PRD) — Smart Age & BD Date
```
## 1. Vision & Strategy
*   **App Name**: Smart Age & BD Date
*   **Category**: Utilities
*   **Core Objective**: Build a highly performant, 100% offline-first application that calculates exact age, birthday countdowns, date differences, and saves local profiles for family and friends.
*   **Target Market**: Global audience with special optimization for South Asian regions (Bangladesh, India, Pakistan) where age calculation forms and public service registration dates are frequently calculated manually.
```
---
```
## 2. Target Personas
```
### Persona A: Tanvir Ahmed (24, Dhaka, Bangladesh)
*   **Role**: Graduate Job Seeker.
*   **Pain Point**: Goverment job application forms in Bangladesh require applicants to input their exact age in years, months, and days on a specific deadline date (e.g., "Age on March 31, 2026").
*   **Usage**: Uses the Date Difference / Government Form Age Helper to compute precise figures instantly.
```
### Persona B: Sarah Collins (32, London, UK)
*   **Role**: Working Mother of Two.
*   **Pain Point**: Forgets extended family birthdays and wants a clean, private, offline tracker to see exactly how many days are left until the next birthday, along with home widgets.
*   **Usage**: Adds family profiles to track birthday countdowns and receives notifications.
```
---
```
## 3. Core Feature List & User Stories
```
### Feature 1: Exact Age Calculator
*   **Requirement**: Calculate exact age based on Date of Birth (DOB) and an optional target date (defaults to Today). Display age in:
    *   Years, Months, Days
    *   Total Months, Total Weeks, Total Days, Total Hours, Total Minutes, and Total Seconds.
*   **User Story**: *As Tanvir, I want to calculate my age as of a specific date so that I can fill out my job application form accurately without manual errors.*
```
### Feature 2: Birthday Countdown & Next Birthday Day
*   **Requirement**: Show days, hours, minutes, and seconds remaining until the next birthday, along with the day of the week the birthday falls on.
*   **User Story**: *As Sarah, I want to see the countdown to my children's birthdays so that I can plan their celebrations ahead of time.*
```
### Feature 3: Birthday Profiles Database (Offline Directory)
*   **Requirement**: Save names and DOBs of family and friends in a local database. Display upcoming birthdays sorted by closest date first.
*   **User Story**: *As Sarah, I want to save all my friends' birthdays in the app so that I have a central, offline registry that alerts me on the day.*
```
### Feature 4: Date Arithmetic & Gap Calculator
*   **Requirement**: Add or subtract days, months, and years from any starting date to find the target date.
*   **User Story**: *As a project planner, I want to add 90 days to a project launch date to calculate the exact deadline.*
```

```
## 4. Key Performance Indicators (KPIs)
*   **Cold Start Time**: < 400ms on mid-range Android devices.
*   **Database Transaction Delay**: Room read/write operations must execute in under 15ms.
*   **Crash Rate**: < 0.1% user-perceived crash rate.
*   **Offline Capability**: 100% functional without cellular or Wi-Fi connectivity.
```
