# 06. AdMob Monetization Map
`
This document outlines AdMob structures for **Breathing Pacer**.
`
---
`
## 1. Placements
*   **Adaptive Banner**: Docked at the bottom of the Configuration Screen.
*   **Interstitial Ad**: Triggered when stopping / finishing a breathing session.
*   **Zero-Ad Focus Zones**: The active breathing visualizer and timer overlays must remain strictly ad-free to preserve relaxation.
`
---
`
## 2. 180s Cooldown Guard
A global cooldown is enforced to prevent interstitial ad spam.
`
```kotlin
package com.saviorsystems.breathingpacer.utils
`
object AdMobHelper {
    private var lastAdShowTime: Long = 0L
    private const val COOLDOWN_MS = 180_000L
`
    fun canShowInterstitial(): Boolean = (System.currentTimeMillis() - lastAdShowTime) >= COOLDOWN_MS
    fun markAdShown() { lastAdShowTime = System.currentTimeMillis() }
}
```
