# 02. UI/UX Design System
`
This document outlines the visual layout, typography, and animation assets for **Breathing Pacer**.
`
---
`
## 1. Color Palette
The primary style uses **Deep Ocean Teal** (`#006064`) to evoke calmness, peace, and slow respiration.
`
### Dark Theme (Default)
*   **Background**: `#0E1416` (Deep Teal Black)
*   **Primary (Brand)**: `#006064` (Deep Ocean Teal)
*   **Secondary (Brand Accent)**: `#00E5FF` (Sky Cyan)
*   **On-Background (Text Primary)**: `#E0F7FA` (Teal Ice White)
*   **On-Background (Text Secondary)**: `#80DEEA` (Muted Cyan)
`
### Light Theme
*   **Background**: `#F2F9F9`
*   **Primary (Brand)**: `#006064`
*   **Secondary (Brand Accent)**: `#00B8D4`
*   **On-Background (Text Primary)**: `#004D40`
*   **On-Background (Text Secondary)**: `#26A69A`
`
---
`
## 2. Typography
*   **Display Title**: `Outfit Bold`, 26sp
*   **Breathing Cue**: `Outfit Medium`, 28sp (Large, centered instruction text)
*   **Pacing Seconds**: `Outfit Bold`, 48sp (Large countdown timer overlay)
*   **Body Copy**: `Inter Regular`, 14sp
`
---
`
## 3. Core Visual Animation
*   **Pacer Sphere**: A circular canvas vector that dynamically scales between `0.3f` (Exhaled/Empty) and `1.0f` (Fully Inhaled/Full) using Compose's animateFloatAsState coupled with linear easing configurations matching cycle durations.
