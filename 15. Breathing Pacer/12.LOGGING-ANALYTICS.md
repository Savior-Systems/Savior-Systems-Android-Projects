# 12. Logging & Analytics
```
This document details logging rules and analytics setup for **Breathing Pacer**.
```
---
```
## 1. Privacy Mandate
*   **Zero PII**: No database exists. No user data, breathing rates, or habits are logged.
```
---
```
## 2. Analytics Events
```
### Event: Breathing Session Completed (`session_completed`)
Fired when a user finishes a pacer session.
*   `preset_type`: `TEXT` ("box", "relax", "custom")
*   `duration_minutes`: `INTEGER` (Time spent, e.g. 5)
```
