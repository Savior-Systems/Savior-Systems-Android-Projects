# 02. UI/UX Design System — Expense Diary Local
`
This design system ensures a premium, trust-evoking, and responsive user interface mapped to the Finance category. The palette is deliberately warm and gold-toned to convey financial confidence and security.
`
---
`
## 1. Color Palette (Material 3 Dynamic Mappings)
`
### Light Mode Color Scheme
*   **Primary (Brand Gold):** `#F1C40F` (Sunflower Gold — represents prosperity and value)
*   **On Primary:** `#1A1A1A` (Deep Charcoal — high contrast text on gold surfaces)
*   **Primary Container:** `#FFF3CD` (Warm Cream — soft card backgrounds)
*   **Secondary:** `#7F8C8D` (Concrete Gray — neutral supporting tone)
*   **Secondary Container:** `#E8EAED` (Silver Mist)
*   **Tertiary (Accent):** `#27AE60` (Emerald Green — income/positive indicators)
*   **Error:** `#E74C3C` (Alizarin Red — expense/negative indicators & deletion)
*   **Background:** `#FAFAF8` (Warm Off-White)
*   **Surface:** `#FFFFFF`
*   **Outline:** `#BDC3C7` (Silver border lines)
`
### Dark Mode Color Scheme
*   **Primary (Brand Gold):** `#F4D03F` (Bright Gold — remains vivid on dark surfaces)
*   **On Primary:** `#1A1A1A`
*   **Primary Container:** `#3D3515` (Deep Amber tint)
*   **Secondary:** `#95A5A6` (Muted Steel)
*   **Tertiary (Accent):** `#2ECC71` (Bright Emerald)
*   **Error:** `#E74C3C`
*   **Background:** `#121212` (Material Dark Base)
*   **Surface:** `#1E1E1E` (Elevated Dark Surface)
*   **Outline:** `#424242`
`

`
## 2. Typography Rules
*   **Primary Font Family:** `Outfit` (Modern, geometric sans-serif, loaded via Google Fonts).
*   **Secondary / Numeric Font:** `DM Mono` (Monospace for financial amounts — ensures digit alignment).
*   **Alternative/Fallback:** `System Default Sans-Serif`.
`
### Size Mappings (Material 3 Type Scale)
| Style | Size | Weight | Usage |
| :--- | :--- | :--- | :--- |
| `Display Large` | 34sp | Bold | Total balance header |
| `Title Large` | 22sp | Semi-Bold | Screen titles, section headings |
| `Title Medium` | 18sp | Medium | Card titles, category names |
| `Body Large` | 16sp | Regular | Transaction notes, descriptions |
| `Body Medium` | 14sp | Regular | List items, secondary text |
| `Label Medium` | 12sp | Medium | Category chips, date labels |
| `Label Small` | 10sp | Regular | Footnotes, disclaimer text |
`
---
`
## 3. Component Design Tokens
`
### 3.1 Transaction Entry Card
*   **Corner Radius**: 16dp (large rounded)
*   **Elevation**: 2dp (subtle lift)
*   **Padding**: 16dp inner, 8dp between fields
*   **Amount Display**: `DM Mono`, 28sp Bold, color-coded (`#27AE60` income / `#E74C3C` expense)
`
### 3.2 Category Icon Grid
*   **Grid Layout**: 4 columns, 16dp gap
*   **Icon Container**: 48dp circle, primary container fill
*   **Selected State**: 3dp gold stroke ring with scale-up spring animation (1.1x)
*   **Icon Style**: Material Icons Rounded, 24dp
`
### 3.3 Budget Progress Bar
*   **Track Height**: 8dp rounded capsule
*   **Fill Gradient**: `#27AE60` → `#F1C40F` → `#E74C3C` (green-to-gold-to-red as usage increases)
*   **Overspend State**: Pulsing red glow animation (2s loop, 0.3 opacity oscillation)
`
### 3.4 FAB (Floating Action Button)
*   **Shape**: Extended FAB with "Add Expense" text + icon
*   **Color**: Primary Gold `#F1C40F` with `#1A1A1A` icon/text
*   **Animation**: Shrinks to standard FAB on scroll down, expands on scroll up
`
---
`
## 4. Gesture Animations & Micro-Interactions
`
### 4.1 Transaction Swipe Actions
*   **Swipe Left (Delete):** Reveals red background with trash icon; spring-back animation on release. Undo Snackbar appears at bottom with 5-second countdown.
*   **Swipe Right (Edit):** Reveals gold background with edit icon; navigates to edit sheet.
`
### 4.2 Amount Entry Keypad
*   **Haptic Feedback:** Light vibration on each key tap (50ms, LOW_AMPLITUDE).
*   **Counter Animation:** Amount text scales up briefly (100ms, 1.05x) on digit entry.
`
### 4.3 Chart Interactions
*   **Pie Chart Tap:** Tapping a slice expands it outward (8dp) and displays the category label with amount overlay.
*   **Bar Chart Scroll:** Horizontal scroll with momentum; snaps to nearest month boundary.
`
### 4.4 Theme Transition
*   **Mode Switch:** Smooth Material 3 crossfade transition (350ms ease-in-out) between light and dark themes.
`
---
`
## 5. Adaptive Layout Rules
*   **Breakpoints:**
    *   Compact (< 600dp): Single-column, bottom navigation.
    *   Medium (600–840dp): Two-column dashboard grid.
    *   Expanded (> 840dp): Three-column with persistent navigation rail.
*   **Orientation Handling:** All composable states survive configuration changes via `rememberSaveable` and ViewModel state persistence.
`
