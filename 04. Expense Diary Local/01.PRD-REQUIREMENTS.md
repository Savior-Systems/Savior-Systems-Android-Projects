# 01. Product Requirement Document (PRD) — Expense Diary Local

## 1. Vision & Strategy
*   **App Name**: Expense Diary Local
*   **Package**: `com.saviorsystems.expensediarylocal`
*   **Category**: Finance
*   **Core Objective**: Build a highly performant, offline-first personal finance management application that empowers users to track daily expenses, set category budgets, and visualize spending patterns — all without requiring an internet connection or sharing data with external servers.
*   **Privacy Promise**: Zero network-dependent features in the core flow. Financial data never leaves the device unless the user explicitly triggers an encrypted local backup export.
*   **Target Keywords**: `expense tracker`, `money manager`, `budget planner`, `spending tracker`, `offline expense diary`, `personal finance`

---

## 2. Target Persona

### 2.1 Primary — The Privacy-Conscious Budgeter
*   **Demographics**: Age 22–45, smartphone-first, budget-aware individuals.
*   **Pain Points**: Distrust of finance apps that require bank login or cloud sync; wants quick entry without friction.
*   **Goal**: Track daily spending in under 5 seconds per entry; review monthly totals at a glance.

### 2.2 Secondary — The Student / Freelancer
*   **Demographics**: Age 18–30, limited income, needs granular category tracking.
*   **Pain Points**: Existing apps are bloated with features they don't need; wants simplicity and speed.
*   **Goal**: Set a monthly budget, get alerted when approaching the limit, export a CSV for tax records.

---

## 3. Core Feature List

### 3.1 Quick Transaction Entry (Main Dashboard)
*   **One-tap Add**: FAB button opens a streamlined bottom sheet for fast expense/income logging.
*   **Fields**: Amount (numeric keypad), Category (icon grid selector), Note (optional text), Date (default today, swipeable date picker).
*   **Transaction Types**: Expense (default) and Income toggle.
*   **Auto-Focus**: Amount field auto-focused on sheet open for zero-friction data entry.

### 3.2 Category Management
*   **Preset Categories**: 12 built-in categories with Material icons:
    *   🍔 Food & Dining, 🚗 Transport, 🛒 Shopping, 🏠 Housing & Rent, 💡 Utilities & Bills, 🎬 Entertainment, 💊 Health & Medical, 📚 Education, ✈️ Travel, 💰 Savings, 🎁 Gifts & Donations, 📦 Miscellaneous.
*   **Custom Categories**: Users can create up to 10 additional custom categories with custom names and icon selection.

### 3.3 Budget Engine
*   **Monthly Budget**: Set an overall monthly spending ceiling.
*   **Category Budgets**: Optional per-category budget limits.
*   **Smart Alerts**: Local push notifications at 80% and 100% budget threshold crossings.
*   **Rollover Toggle**: Option to carry unspent budget into the next month.

### 3.4 Analytics Dashboard
*   **Spending Overview**: Interactive donut/pie chart showing category-wise expense distribution.
*   **Monthly Bar Chart**: Horizontal bar chart comparing month-over-month expense trends.
*   **Summary Cards**: Total Income, Total Expenses, Net Balance, Top Spending Category.
*   **Date Range Filter**: Quick presets (This Week, This Month, Last 30 Days, Custom Range).

### 3.5 Transaction History
*   **Chronological List**: Grouped by date with daily subtotals.
*   **Search & Filter**: Full-text search on notes, filter by category, type (income/expense), and date range.
*   **Swipe Actions**: Swipe-left to delete (with undo snackbar), swipe-right to edit.
*   **Pagination**: Paging 3 integration for smooth scrolling through large datasets.

### 3.6 Data Export & Backup
*   **CSV Export**: Export filtered transaction history as a `.csv` file to device storage.
*   **PDF Report**: Generate a monthly summary PDF with charts embedded.
*   **JSON Backup/Restore**: Full database backup to a local `.json` file; restore from backup file picker.

### 3.7 Security & Privacy
*   **App Lock**: Optional PIN (4–6 digit) or Biometric (fingerprint/face) authentication on app launch.
*   **Privacy Screen**: Auto-blur amounts when the app enters the background (prevent shoulder-surfing on recent apps).
*   **SQLCipher Encryption**: Full-database AES-256 encryption at rest using Android Keystore-managed keys.

### 3.8 Settings & Personalization
*   **Currency Selector**: 30+ currencies with locale-aware formatting.
*   **Theme Toggle**: System Default / Light / Dark mode with Material 3 dynamic theming.
*   **First Day of Week**: Configurable (Sunday/Monday) for budget cycle alignment.
*   **Data Management**: Clear all data, export, and restore options.
