# 05. Database Schema & Room DAO — Routine Widget
`
Database schemas managing daily schedules and tracking task state flags.
`
---
`
## 1. Entity: `routine_tasks`
`
This table stores the template tasks that recur every week.
`
| Column Name | SQLite Data Type | Constraints / Keys | Description |
| :--- | :--- | :--- | :--- |
| `task_id` | INTEGER | PRIMARY KEY AUTOINCREMENT | Primary Key |
| `task_name` | TEXT | NOT NULL | Title of the task |
| `target_time` | TEXT | NOT NULL | Scheduled time string (e.g., `09:00 AM`) |
| `day_of_week` | INTEGER | NOT NULL | Target day index (1 = Mon, 7 = Sun) |
| `is_completed_today` | INTEGER | DEFAULT 0 | 0 = Pending, 1 = Done |
`
---
`
## 2. Entity: `completion_logs`
`
Saves history statistics to calculate daily consistency scores.
`
| Column Name | SQLite Data Type | Constraints / Keys | Description |
| :--- | :--- | :--- | :--- |
| `log_id` | INTEGER | PRIMARY KEY AUTOINCREMENT | Primary Key |
| `log_date` | TEXT | NOT NULL | Date string (`YYYY-MM-DD`) |
| `tasks_completed` | INTEGER | NOT NULL | Count of tasks completed |
| `total_tasks` | INTEGER | NOT NULL | Total tasks configured for that day |
`
---
`
## 3. Room DAO Interface
`
```kotlin
package com.saviorsystems.routinewidget.data.database
`
import androidx.room.*
import kotlinx.coroutines.flow.Flow
`
@Dao
interface RoutineDao {
`
    @Query("SELECT * FROM routine_tasks WHERE day_of_week = :dayIndex ORDER BY target_time ASC")
    fun getTasksForDayFlow(dayIndex: Int): Flow<List<RoutineTaskEntity>>
`
    @Insert(onConflict = OnConflictStrategy.REPLACE)
    suspend fun insertTask(task: RoutineTaskEntity): Long
`
    @Query("UPDATE routine_tasks SET is_completed_today = :isCompleted WHERE task_id = :taskId")
    suspend fun updateTaskStatus(taskId: Long, isCompleted: Boolean)
`
    @Query("UPDATE routine_tasks SET is_completed_today = 0")
    suspend fun resetAllDailyTasks()
`
    @Delete
    suspend fun deleteTask(task: RoutineTaskEntity)
`
    @Insert(onConflict = OnConflictStrategy.REPLACE)
    suspend fun insertCompletionLog(log: CompletionLogEntity)
}
```
