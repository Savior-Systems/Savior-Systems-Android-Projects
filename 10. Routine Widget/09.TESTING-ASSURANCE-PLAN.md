## 2. Manual QA Matrix (Edge Cases)
```
| QA Case | Action | Expected Behavior |
| :--- | :--- | :--- |
| **Midnight Rollover** | Set device clock to `23:59`, launch app, and wait. | Checkboxes must reset to unchecked status at `00:00` and display the new day's template. |
| **Fast Double-Clicks** | Tap checkbox button repeatedly in rapid succession. | Widget UI updates once; DB handles requests atomicly without concurrent-write crashes. |
| **No Tasks Set** | Place a new widget on a clean install. | Renders empty state messaging and deep-links to configurator on click. |
| **Widget Host Limit** | Place 3 widgets on different home panels. | Checking off a task on widget A updates widget B and C immediately. |
```
---
```
## 3. Performance & Sync Benchmarks
*   **Editor Boot Speed**: `< 400ms` cold start.
*   **Widget Click Sync**: Tapping widget checkbox must commit to Room and refresh widget view in `< 150ms`.
*   **Worker Overhead**: WorkManager midnight reset task must complete in `< 500ms`.
