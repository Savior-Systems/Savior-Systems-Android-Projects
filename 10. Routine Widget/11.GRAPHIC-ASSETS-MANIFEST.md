## 2. Adaptive Icon Layer Layout
```
Built matching Material 3 design systems:
*   **Background Layer (`res/drawable/ic_launcher_background.xml`)**: Solid hex `#E65100` (Sunset Orange) background.
*   **Foreground Layer (`res/drawable/ic_launcher_foreground.xml`)**: Vector path showing a clean desk calendar grid with an active checkbox marker in white and gold accents.
*   **Monochrome Layer (`res/drawable/ic_launcher_monochrome.xml`)**: Clean vector outline allowing dynamic system-wide theme coloring.
```
