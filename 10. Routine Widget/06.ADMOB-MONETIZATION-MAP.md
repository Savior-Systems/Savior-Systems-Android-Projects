# 06. AdMob Monetization Map — Routine Widget

Monetization maps tailored to protect the widget interfaces and avoid user interference.

---

## 1. Ad Format Placements

| Format | Screen Target | Placement Details |
| :--- | :--- | :--- |
| **Banner** | Task Configurator Activity | Anchored at the bottom of the editor. |
| **Interstitial** | Save Template Screen | Loaded and shown when the user finishes configuring their weekly schedule. |

---

## 2. Zero-Ad Zones (100% Ad Free)
*   **Android Home Screen Widgets**: Complying with Android system design standards, **zero ads** are allowed inside the widget canvas itself.
*   **Task Checkoff Action**: Tapping widget checkboxes to complete tasks must never trigger popups or interstitial interruptions.

---

## 3. Interstitial Cooldown Limits
A 180-second throttle timer ensures users editing routines back-to-back are not subject to excessive ad loads.

```kotlin
object RoutineAdManager {
    private const val AD_COOLDOWN_INTERVAL_MS = 180_000L // 3 minutes
    private var lastAdShowTime = 0L

    fun shouldShowInterstitial(): Boolean {
        return (System.currentTimeMillis() - lastAdShowTime) >= AD_COOLDOWN_INTERVAL_MS
    }

    fun markAdShown() {
        lastAdShowTime = System.currentTimeMillis()
    }
}
```
*   **Workflow**: If `shouldShowInterstitial()` resolves to false when saving the schedule, transition directly back to the main list without fetching an interstitial.
