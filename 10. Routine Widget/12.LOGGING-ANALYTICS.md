## 1. Firebase Custom Events
```
### Event A: `widget_checkbox_clicked`
*   **Trigger**: User marks a task complete from the home screen widget.
*   **Parameters**:
    *   `is_completed` (integer: 1 or 0)
```
### Event B: `routine_saved`
*   **Trigger**: User completes a routine edit sequence.
*   **Parameters**:
    *   `tasks_count` (integer: total tasks saved)
    *   `day_index` (integer: Mon-Sun)
```
### Event C: `midnight_reset_success`
*   **Trigger**: WorkManager midnight reset completes.
*   **Parameters**:
    *   `duration_ms` (integer: execution time)
```
### Event D: `app_reset_triggered`
*   **Trigger**: User purges local schedule database via settings.
*   **Parameters**:
    *   `cleared_tasks_count` (integer)
```

```
## 2. Privacy Guardrails ("NEVER Log" Rules)
*   🚫 **NEVER log Task Names**: Descriptions or custom names entered by the user (e.g. "Meeting with John") must never be sent to analytics parameters.
*   🚫 **NEVER log Time offsets**: Do not log specific timezone structures or user schedule times (e.g., `09:12 AM`).
*   🚫 **NEVER log Widget locations**: Do not track host panels or page details of the launcher app screen.
