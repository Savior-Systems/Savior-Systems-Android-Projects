# 03. Functional Flows & Navigation

## 1. Overall App Flowchart

```text
[ App Launch ]
      │
      ▼
[ Permissions Check ] ──(Denied)──> [ Rationale Dialog ]
      │(Granted)
      ▼
[ Dashboard Screen ]
  - List of past recordings
  - Huge "Record" FAB at bottom
      │
      ├─> (Tap Record) ──> Starts Foreground Service ──> [ Recording Screen ]
      │                                                     - Timer (00:00:00)
      │                                                     - Pulse Visualizer
      │                                                     - Stop Button
      │                                                           │
      │                                                           ▼
      │                                                    [ Save Dialog ]
      │                                                 (Prompt for Title)
      │
      └─> (Tap Item in List) ──> [ Playback & Summary Screen ]
                                    - Play/Pause, Seekbar
                                    - Existing AI Summary Text
                                    - "Generate AI Summary" Button
```

## 2. AI Summarization Flow (The Vertex AI Flow)
1. User is on the Playback screen.
2. User taps "Generate AI Summary".
3. **Ad Trigger:** Show Rewarded Video Ad. "Watch Ad to generate summary".
4. If ad watched -> Show loading spinner.
5. Upload local URI to Vertex AI Gemini 1.5 Flash.
6. Await response -> Update Room DB with text -> Display text in UI.
