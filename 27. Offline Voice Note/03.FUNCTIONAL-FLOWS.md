# 03. Functional Flows

## App Launch & Onboarding
1. Splash Screen (App Icon + Brand color).
2. GDPR UMP Consent Dialog (EU only).
3. Permission Request (Only if absolutely necessary).
4. Home Screen.

## Core Loop
1. User opens Offline Voice Note Recorder.
2. User executes the primary utility (Record audio memos, dictations, and voice notes fully offline without transcription tracking.).
3. Interstitial Ad displayed on completion (subject to 180s cooldown).
4. Data persisted locally via Room DB (Audio Metadata).
