# 05. Database Schema & Storage

## 1. Table: `voice_notes`
Since the actual audio is a `.m4a` file stored in the app's internal storage (`Context.filesDir`), Room only stores the metadata.

| Column          | Type    | Constraints                  | Description |
|-----------------|---------|------------------------------|-------------|
| `id`            | Long    | Primary Key, Auto-Increment  | Unique ID   |
| `title`         | String  | Not Null                     | "Lecture 1" |
| `filePath`      | String  | Not Null                     | Absolute path to `.m4a` |
| `durationMs`    | Long    | Not Null                     | Audio duration |
| `createdAt`     | Long    | Not Null                     | Epoch timestamp |
| `aiTranscription`| String | Nullable                     | The raw text from Gemini |
| `aiSummary`     | String  | Nullable                     | The bullet points from Gemini |

## 2. DAO Queries
```kotlin
@Query("SELECT * FROM voice_notes ORDER BY createdAt DESC")
fun getAllNotes(): Flow<List<VoiceNoteEntity>>

@Insert
suspend fun insertNote(note: VoiceNoteEntity)

@Query("UPDATE voice_notes SET aiTranscription = :transcript, aiSummary = :summary WHERE id = :id")
suspend fun updateAiData(id: Long, transcript: String, summary: String)
```
