# 04. Technical Architecture Blueprint

## 1. Architecture Choice: MVI / MVVM
- **UI Layer:** Jetpack Compose.
- **Service Layer:** `VoiceRecorderService` extending `Service`. This is required to keep recording alive when the app is in the background.
- **AI Layer:** Vertex AI in Firebase SDK (`com.google.firebase:firebase-vertexai`).

## 2. Recommended Tech Stack
- **Audio Engine:** `MediaRecorder` for recording, `ExoPlayer` or `MediaPlayer` for playback.
- **Local Database:** Room Database.
- **Dependency Injection:** Hilt.

## 4. Vertex AI Architecture
- Use `GenerativeModel("gemini-1.5-flash")`.
- Pass the audio file natively. Gemini 1.5 Flash can process audio tokens.
- **Prompt:** "You are an expert transcriptionist. Please transcribe this audio exactly. Then, provide a 3-bullet point summary of the main topics."
