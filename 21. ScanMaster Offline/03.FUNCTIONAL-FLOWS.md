# 03. Functional Flows

## App Launch & Onboarding
1. Splash Screen (App Icon + Brand color).
2. GDPR UMP Consent Dialog (EU only).
3. Permission Request (Only if absolutely necessary).
4. Home Screen.

## Core Loop
1. User opens ScanMaster Document Scanner.
2. User executes the primary utility (Offline PDF document scanner using CameraX and ML Kit Document Scanner API.).
3. Interstitial Ad displayed on completion (subject to 180s cooldown).
4. Data persisted locally via Room DB (Scanned Docs, PDF paths).
