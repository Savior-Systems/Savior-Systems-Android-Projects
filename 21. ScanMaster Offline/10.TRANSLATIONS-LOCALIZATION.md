# 10. Translations & Localization

## Default Language
`en-US` (English)

## Tier 1 Expansions
- `es-ES` (Spanish)
- `pt-BR` (Portuguese - Brazil)
- `bn-BD` (Bengali)
- `hi-IN` (Hindi)

## Implementation
All hardcoded strings must be extracted to `res/values/strings.xml`. Absolutely no raw text in Compose files.
