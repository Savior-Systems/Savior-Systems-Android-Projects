## 1. Entity: `saved_calculations`
```
This table stores calculations saved by the user for comparison.
```
| Column Name | SQLite Data Type | Constraints / Keys | Description |
| :--- | :--- | :--- | :--- |
| `calc_id` | INTEGER | PRIMARY KEY AUTOINCREMENT | Primary Key |
| `title` | TEXT | NOT NULL | Custom user label (e.g. "My Tax FY26") |
| `calc_type` | TEXT | NOT NULL | `"TAX"` or `"VAT"` |
| `input_value` | REAL | NOT NULL | Raw input amount (salary or invoice base) |
| `output_result` | REAL | NOT NULL | Total output (Tax payable or VAT amount) |
| `exemption_tier` | TEXT | NULLABLE | Selected tax exemption category |
| `parameters_json` | TEXT | NOT NULL | Raw JSON mapping of slab results / VAT rates |
| `created_at` | INTEGER | NOT NULL | Epoch millisecond timestamp |
```

## 2. Room DAO Interface
```
```kotlin
package com.saviorsystems.bdtaxcalc.data.database
```
import androidx.room.*
import kotlinx.coroutines.flow.Flow
```
@Dao
interface CalculationDao {
```
    @Query("SELECT * FROM saved_calculations ORDER BY created_at DESC")
    fun getAllCalculationsFlow(): Flow<List<CalculationEntity>>
```
    @Insert(onConflict = OnConflictStrategy.REPLACE)
    suspend fun insertCalculation(calc: CalculationEntity): Long
```
    @Delete
    suspend fun deleteCalculation(calc: CalculationEntity)
```
    @Query("DELETE FROM saved_calculations")
    suspend fun clearHistory()
}
