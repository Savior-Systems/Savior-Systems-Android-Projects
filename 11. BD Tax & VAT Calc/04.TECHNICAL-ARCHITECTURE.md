## 2. Core Tax Calculation Engine (Kotlin)
```
This engine computes progressive tax liability dynamically by reading slab bounds from configurations, protecting the app from hardcode legacy limits when the July budget changes.
```
```kotlin
package com.saviorsystems.bdtaxcalc.engine
```
import com.saviorsystems.bdtaxcalc.data.model.TaxSlabConfig
import com.saviorsystems.bdtaxcalc.data.model.TaxSlabRule
```
object TaxCalculationEngine {
```
    data class SlabResult(
        val ratePercentage: Int,
        val taxableAmountInSlab: Double,
        val taxCharged: Double
    )
```
    data class TaxBreakdown(
        val totalTaxableIncome: Double,
        val taxFreeExemptionLimit: Double,
        val totalTaxPayable: Double,
        val slabsBreakdown: List<SlabResult>
    )
```
    fun calculateTax(
        taxableIncome: Double,
        exemptionLimit: Double,
        slabRules: List<TaxSlabRule>,
        minimumTaxLimit: Double = 3000.0
    ): TaxBreakdown {
        val breakdownList = mutableListOf<SlabResult>()
```
        if (taxableIncome <= exemptionLimit) {
            return TaxBreakdown(taxableIncome, exemptionLimit, 0.0, emptyList())
        }
```
        var remainingIncome = taxableIncome - exemptionLimit
        var totalCalculatedTax = 0.0
```
        for (rule in slabRules) {
            if (remainingIncome <= 0) break
```
            val slabCapacity = rule.limitAmount // e.g. 300000.0 or Double.MAX_VALUE
            val amountInThisSlab = if (remainingIncome > slabCapacity) slabCapacity else remainingIncome
            val taxInThisSlab = amountInThisSlab * (rule.ratePercentage / 100.0)
```
            breakdownList.add(
                SlabResult(
                    ratePercentage = rule.ratePercentage,
                    taxableAmountInSlab = amountInThisSlab,
                    taxCharged = taxInThisSlab
                )
            )
```
            totalCalculatedTax += taxInThisSlab
            remainingIncome -= amountInThisSlab
        }
```
        // Apply geographical minimum tax rules if total tax > 0
        val finalTax = if (totalCalculatedTax > 0.0 && totalCalculatedTax < minimumTaxLimit) {
```minimumTaxLimit
        } else {
```totalCalculatedTax
        }
```
        return TaxBreakdown(
            totalTaxableIncome = taxableIncome,
            taxFreeExemptionLimit = exemptionLimit,
            totalTaxPayable = finalTax,
            slabsBreakdown = breakdownList
        )
    }
}

## 3. Dynamic JSON Asset Config Schema (`assets/tax_rules.json`)
Allows instant July budget updates:
```json
{
  "fiscal_years": {
    "2025-2026": {
      "rules": [
        {"limitAmount": 300000.0, "ratePercentage": 10},
        {"limitAmount": 400000.0, "ratePercentage": 15},
        {"limitAmount": 500000.0, "ratePercentage": 20},
        {"limitAmount": 2000000.0, "ratePercentage": 25},
        {"limitAmount": 999999999.0, "ratePercentage": 30}
      ]
    }
  }
}
```
*The engine reads this file on load, meaning the developer can simply copy-paste a new fiscal year block when NBR announces updates in July.*
