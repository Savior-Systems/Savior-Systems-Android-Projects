# 02. UI/UX Design System — BD Tax & VAT Calc
```
Visually clean, trust-evoking finance theme utilizing the Forest Green color palette.
```
---
```
## 1. Color System (Forest Green)
```
### Light Theme
*   **Primary (`#2E7D32`)**: Rich Forest Green representing finance, growth, and security. Used for headers, FABs, and active sliders.
*   **Primary Container (`#E8F5E9`)**: Light mint container backgrounds.
*   **Secondary (`#F57F17`)**: Warm Gold/Amber accent for tax savings highlight and alert buttons.
*   **Tertiary (`#37474F`)**: Slate gray for card borders.
*   **Background (`#FBFDFB`)**: Soft off-white page background.
*   **Surface (`#FFFFFF`)**: Pure white cards for forms and result grids.
*   **On-Background (`#1A1E1A`)**: Deep forest charcoal text.
```
### Dark Theme
*   **Primary (`#81C784`)**: Soft pastel green accent.
*   **Primary Container (`#1B5E20`)**: Deep dark forest green containers.
*   **Secondary (`#FFF59D`)**: Pastel gold for savings alerts.
*   **Tertiary (`#90A4AE`)**: Cool slate.
*   **Background (`#0F120F`)**: Dark forest-black.
*   **Surface (`#171B17`)**: Card backings.
*   **On-Background (`#E8EBE8`)**: Light grey text.
```
---
```
## 2. Typography Standard
*   **Font Family**: **Outfit** (via Google Fonts Integration) for geometric financial figures.
*   **Display Large (Calculated Tax)**: 40sp, Bold, tracking -0.5px.
*   **Title Large (Card Header)**: 18sp, Semi-Bold, tracking 0.15px.
*   **Body Medium (Labels)**: 14sp, Regular.
```
---
```
## 3. Micro-Animations
```
### Animation A: Slab Accordion Slide
*   **Trigger**: Expanding the detailed tax slab breakdown card.
*   **Behavior**: Smooth height expansion via `tween` spec (300ms, `FastOutSlowInEasing`) with a minor fade-in alpha slide of each slab item to highlight progressive calculation steps.
```
### Animation B: Value Counter Spring
*   **Trigger**: Calculating results.
*   **Behavior**: The total tax and VAT numbers perform a numeric roll-up animation (spring scaling from 0.98 to 1.0) over 400ms to mimic cash counter transitions.
```
