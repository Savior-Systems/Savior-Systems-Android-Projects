## 1. Ad Format Placements
```
| Format | Screen Target | Placement Details |
| :--- | :--- | :--- |
| **Banner** | Calculator Bottom | Anchored at the bottom below form inputs. |
| **Native Ad** | History List Layout | Integrated inline as a list card after every 4 saved calculations. |

## 2. Zero-Ad Zones
*   **Data Entry Input Fields**: No banner or popup ads loaded near active numeric keyboard zones to prevent accidental clicks.
*   **GDPR Consent Form**: SDK loading blocked until UMP consent resolves.

## 3. Strict Interstitial Cooldown
```
To avoid spamming users who adjust income sliders in rapid succession:
```kotlin
object TaxAdManager {
    private const val AD_COOLDOWN_INTERVAL_MS = 180_000L // 3 minutes
    private var lastAdShowTime = 0L
```
    fun shouldShowInterstitial(): Boolean {
        return (System.currentTimeMillis() - lastAdShowTime) >= AD_COOLDOWN_INTERVAL_MS
    }
```
    fun markAdShown() {
        lastAdShowTime = System.currentTimeMillis()
    }
}
```
*   **Usage**: On click "Calculate", check `shouldShowInterstitial()`. If false, skip the ad and display results immediately.
