# 09. Testing & Quality Assurance Plan — BD Tax & VAT Calc
`
Verifying mathematical accuracy for progressive tax bands, location minimum overrides, and VAT formulas.
`
---
`
## 1. Automated Test Cases
*   **Unit Tests (`TaxCalculationEngineTest.kt`)**:
    *   Test baseline exemption limit checks (e.g. BDT 375,000 general slab yields BDT 0 tax).
    *   Assert math logic for progressive slabs (e.g. BDT 1,075,000 yields exactly BDT 90,000 tax).
    *   Assert female threshold (exemption limit BDT 425,000).
    *   Assert disability/third gender limits (exemption BDT 500,000).
    *   Assert war-wounded freedom fighters limit (exemption BDT 525,000).
*   **Unit Tests (`VatCalculationEngineTest.kt`)**:
    *   VAT exclusive formula output.
    *   VAT inclusive formula output.
`
---
`
## 2. Manual QA Matrix (Edge Cases)
`
| QA Case | Action | Expected Behavior |
| :--- | :--- | :--- |
| **Minimum Tax Override** | Input taxable income yielding BDT 1,200 progressive tax in Dhaka. | Override output to display local minimum tax: BDT 5,000. |
