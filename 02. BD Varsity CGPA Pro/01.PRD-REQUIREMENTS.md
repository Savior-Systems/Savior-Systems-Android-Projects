# 01. Product Requirement Document (PRD)
```
## 1. Vision & Strategy
```
*   **App Name**: BD Varsity CGPA Pro
*   **Package Name**: `com.saviorsystems.education.bdcgpa`
*   **Category**: Education
*   **Core Objective**: Build a lightweight, offline-first CGPA calculator that provides pre-loaded grading scales for 17+ Bangladeshi public and private universities — eliminating the need for students to manually look up grade-point mappings or perform repetitive mental math during results season.
*   **Primary Keywords**: cgpa calculator, nu cgpa, dhaka varsity gpa, buet grading, bd university gpa calculator, semester gpa calculator
*   **Unique Selling Proposition (USP)**: The only offline CGPA calculator with a **built-in library of BD university grading presets** — DU, NU, BUET, BRAC, NSU, and 12+ more — so students can calculate accurate CGPAs without needing to know their university's specific grade-point scale.
```

## 2. Target Personas
```
### Persona A — Rahim (National University Honours Student)
*   **Age**: 20 · **Location**: Mymensingh, Bangladesh
*   **Device**: Mid-range Android (Redmi Note series) · API 28-30
*   **Context**: National University has 2.2M+ students across 2,200+ affiliated colleges. Rahim studies Honours in Political Science. After each semester, he receives letter grades and needs to calculate his CGPA manually using NU's grading scale (A+ = 4.00, A = 3.75, ... D = 2.00). He currently uses a calculator app and a paper chart.
*   **Pain Point**: Doesn't remember the exact grade-point mapping for B- vs C+. Needs a quick way to enter his 6-8 courses per semester and see the result.
*   **Goal**: Instant CGPA with NU's grading scale pre-loaded. Bangla UI preferred.
```
### Persona B — Tasnia (BRAC University Undergrad)
*   **Age**: 21 · **Location**: Dhaka, Bangladesh
*   **Device**: Samsung Galaxy A series · API 33
*   **Context**: Studies Computer Science at BRAC University, which uses a different grading scale (A = 4.00 starts at 90%, unlike DU's 80%). She takes 4-5 courses per semester with varying credit hours (1.5 to 3.0 credits).
*   **Pain Point**: Existing generic GPA calculators don't know BRAC's specific scale. She has to manually enter grade-point values every time.
*   **Goal**: Select "BRAC University" once and just enter letter grades. Track multiple semesters.
```
### Persona C — Shuvo (BUET Engineering Student)
*   **Age**: 22 · **Location**: Dhaka, Bangladesh
*   **Device**: Xiaomi Poco · API 31
*   **Context**: BUET uses continuous assessment (quizzes 30% + finals 70%) with its own grade scale. Shuvo wants to predict his CGPA mid-semester by estimating grades.
*   **Pain Point**: Wants to simulate "what if I get B+ in this course?" scenarios without redoing the entire calculation.
*   **Goal**: Quick course-by-course grade entry with live recalculation. English UI. Share result as text to WhatsApp group.
```

## 4. User Stories
```
| ID | Story | Acceptance Criteria |
|:---|:------|:--------------------|
| US-01 | As a student, I want to select my university from a preset list so that I don't have to manually look up my grading scale. | University selector shows 17+ options. Selection persists across app restarts. |
| US-02 | As a student, I want to add courses to a semester with credit hours and letter grades so that I can calculate my GPA. | Course card shows name field, credit input (0.5-6.0), grade dropdown. At least 1 course required to calculate. |
| US-03 | As a student, I want to see my CGPA update in real-time as I change grades so that I can simulate scenarios. | CGPA display updates within 100ms of any grade or credit change. No button press needed. |
| US-04 | As a student, I want to manage multiple semesters so that I can track my cumulative CGPA across my degree. | Add/rename/delete semesters. Cumulative CGPA spans all semesters. |
| US-05 | As a student, I want to view my university's grading table so that I know which letter grade maps to which grade point. | Table shows letter, GP, and percentage range. Matches selected university. |
| US-06 | As a student, I want to save my calculation results so that I can review them later without re-entering data. | Saved results show university, date, semester count, total credits, CGPA. Tappable to view details. |
| US-07 | As a student, I want to share my CGPA as text so that I can send it to friends or family on WhatsApp. | Share produces formatted text: "My CGPA: 3.67 (DU Scale) — 6 Semesters, 120 Credits". |
| US-08 | As a student at an unlisted university, I want to create a custom grading scale so that I can still use the calculator. | Custom university form: name, add letter grades (letter, GP). Saved to database. |
| US-09 | As a user, I want to switch the app to Bangla so that I can use it in my native language. | All UI text switches to Bangla. University names show Bangla variants. Numbers remain Latin digits. |
| US-10 | As a user, I want the app to work fully offline so that I can use it during load-shedding or in areas with poor connectivity. | All features functional in airplane mode. No network requests for core functionality. |
```

## 6. Scope Boundaries
```
### ✅ In Scope
*   Offline CGPA/GPA calculation with pre-loaded BD university grading scales
*   Local storage of calculation history
*   Bilingual UI (English + Bangla)
*   Custom grading scale creation
*   AdMob monetization (banner + interstitial)
```
### ❌ Out of Scope
*   Online transcript syncing or verification with university systems
*   Official grade submission or registration
*   GPA-to-percentage conversion (different problem domain)
*   Push notifications or reminders
*   User account registration or cloud sync
*   Grade prediction or AI-based recommendations
```
