## 3. Component Library
```
### 3.1 CGPA Result Display Card
The hero component at the top of the calculator screen.
```
```
┌─────────────────────────────────────┐
│         Your CGPA                   │
│                                     │
│     ┌─────────────────────┐         │
│     │                     │         │
│     │       3.67          │  ← displayLarge, primary color
│     │     ──────          │  ← divider
│     │   out of 4.00       │  ← bodyMedium, onSurfaceVariant
│     │                     │         │
│     └─────────────────────┘         │
│                                     │
│  Total Credits: 120  │  6 Semesters │  ← bodyMedium chips
│  University: DU                     │  ← labelMedium
│                                     │
│  [ 📤 Share Result ]                │  ← outlined button, tertiary
└─────────────────────────────────────┘
```
*   **Shape**: RoundedCornerShape(16.dp)
*   **Elevation**: 2.dp (tonalElevation)
*   **Background**: surfaceVariant
*   **CGPA Number**: Uses `displayLarge` with animated counter (count-up from 0.00)
*   **Padding**: 24.dp all sides
```
### 3.2 Course Entry Card
Repeating card for each course within a semester.
```
```
┌─────────────────────────────────────┐
│  Course Name (optional)      [ 🗑️ ] │  ← OutlinedTextField + IconButton
│                                     │
│  Credit Hours: [  3.0  ]            │  ← NumberField (0.5 - 6.0, step 0.5)
│                                     │
│  Grade:  [ A+ (4.00) ▼ ]            │  ← ExposedDropdownMenu from university scale
│                                     │
│  Grade Point: 4.00 × 3.0 = 12.00   │  ← Computed, bodyMedium, onSurfaceVariant
└─────────────────────────────────────┘
```
*   **Shape**: RoundedCornerShape(12.dp)
*   **Border**: 1.dp outline color
*   **Padding**: 16.dp
*   **Spacing**: 8.dp between fields
```
### 3.3 Semester Header (Collapsible)
```
┌─────────────────────────────────────┐
│  ▼ Semester 1          GPA: 3.75    │  ← clickable, expand/collapse
│     4 courses · 12.0 credits        │  ← bodySmall, onSurfaceVariant
└─────────────────────────────────────┘
```
*   **Shape**: RoundedCornerShape(8.dp)
*   **Background**: primaryContainer
*   **Content Color**: onPrimaryContainer
*   Tap to expand/collapse course list
*   Long-press to rename or delete

## 4. Spacing & Layout Grid
```
| Token | Value | Usage |
|:------|:------|:------|
| `spacing_xs` | 4.dp | Inline spacing between small elements |
| `spacing_sm` | 8.dp | Between related items (e.g., label and input) |
| `spacing_md` | 12.dp | Card internal padding |
| `spacing_lg` | 16.dp | Screen horizontal padding, card gaps |
| `spacing_xl` | 24.dp | Section separators, hero card padding |
| `spacing_xxl` | 32.dp | Major section breaks |
| `screen_padding` | 16.dp | Consistent horizontal screen padding |
| `card_corner` | 12.dp | Standard card corner radius |
| `hero_corner` | 16.dp | Hero card corner radius |
| `bottom_bar_height` | 56.dp | Banner ad + padding space |
```

## 6. Animation Specifications
```
| Animation | Type | Duration | Easing |
|:----------|:-----|:---------|:-------|
| CGPA counter (0.00 → result) | `animateFloatAsState` | 800ms | FastOutSlowIn |
| Course card enter | `AnimatedVisibility` + `fadeIn` + `slideInVertically` | 300ms | EaseOut |
| Course card exit (delete) | `AnimatedVisibility` + `fadeOut` + `shrinkVertically` | 250ms | EaseIn |
| Semester expand/collapse | `AnimatedVisibility` + `expandVertically` | 300ms | FastOutSlowIn |
| Bottom sheet slide | `ModalBottomSheet` default | 300ms | Standard |
| Screen transitions | `NavHost` composable transitions | 300ms | FastOutSlowIn |
| FAB appearance | `AnimatedVisibility` + `scaleIn` | 200ms | EaseOut |
```

## 7. Responsive Behavior
```
*   **Phone Portrait** (default): Single-column layout. Full-width cards.
*   **Phone Landscape**: Lock to portrait orientation. CGPA calculator benefits from vertical scrolling.
*   **Tablet** (optional, v2): Two-column layout — semesters list on left, course entry on right.
```
### Configuration
```kotlin
android {
    defaultConfig {
        // Lock to portrait for v1
        screenOrientation = ActivityInfo.SCREEN_ORIENTATION_PORTRAIT
    }
}
```
