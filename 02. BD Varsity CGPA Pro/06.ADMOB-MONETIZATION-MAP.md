## 4. Ad Placement Rules
`
### 4.1 Adaptive Banner
*   **Screen**: Main Calculator Screen only.
*   **Position**: Anchored at the very bottom of the screen, below all scrollable content.
*   **Size**: `AdSize.ANCHORED_ADAPTIVE` (responsive to screen width).
*   **Loading**: Load on screen creation. Auto-refresh every 45 seconds (AdMob default).
*   **Constraint**: Must not overlap with the "Add Semester" FAB or any interactive buttons. Minimum 24dp spacing between the banner and the nearest tappable element.
`
### 4.2 Interstitial
*   **Trigger**: Shown ONLY after the user taps "Save Result" (💾) and the result is successfully written to the database.
*   **Cooldown**: Hard 180-second frequency cap enforced by `AdManager`.
*   **Flow**:
    1. User taps Save → ViewModel saves to Room DB
    2. Check cooldown: `(currentTime - lastShowTime) >= 180_000L`
    3. If eligible → show interstitial → on dismiss → show "Saved!" snackbar
    4. If cooldown active → skip ad → show "Saved!" snackbar immediately
*   **Pre-loading**: Interstitial is pre-loaded on Calculator screen creation and after each dismissal.
`
### 4.3 Prohibited Placements
`
| ❌ Never Show Ads Here | Reason |
|:-----------------------|:-------|
| During course/grade input | Interrupts typing flow, accidental clicks |
| On University Selection screen | First-time user experience — no ads |
| During CGPA calculation animation | Disrupts core value delivery |
| On Back press | Violates AdMob policy |
| On Splash screen | Too early in session, policy risk |
| On Settings/About screens | Low-value placement, annoys users |
`

## 5. UMP Consent Integration
`
*   Initialize Google UMP (User Messaging Platform) SDK in `App.kt` before any ad request.
*   GDPR consent dialog triggers automatically for users in the EEA/UK.
*   If consent is denied → fall back to non-personalized ads.
*   Consent status is cached and rechecked every 30 days.
`
```kotlin
// In App.kt onCreate()
val params = ConsentRequestParameters.Builder()
    .setTagForUnderAgeOfConsent(false)
    .build()
`
ConsentInformation.getInstance(this).requestConsentInfoUpdate(
    this as Activity, params, { /* ready to load ads */ }, { /* handle error */ }
)
```
