# 11. Graphic Assets Manifest

This document specifies all visual assets required for **BD Varsity CGPA Pro**, including the app icon, Play Store graphics, and screenshot requirements.

---

## 1. App Icon

### Brand Identity
*   **Concept**: Graduation cap (🎓) combined with a calculator motif, representing academic calculation
*   **Primary Color**: Emerald Green `#2ECC71`
*   **Background Color**: Wet Asphalt `#34495E`
*   **Style**: Flat design with subtle depth. Clean, modern, and recognizable at small sizes.

### Adaptive Icon Structure
```
┌─────────────────────────────┐
│      Foreground Layer       │
│                             │
│    ┌───────────────────┐    │
│    │   🎓               │    │
│    │   ┌─────────┐     │    │  ← Graduation cap on top
│    │   │ = 4.00  │     │    │  ← Calculator display below
│    │   │ ─────── │     │    │
│    │   │ [1][2][3]│     │    │  ← Subtle calculator keys
│    │   └─────────┘     │    │
│    └───────────────────┘    │
│                             │
│      (transparent bg)       │
└─────────────────────────────┘

┌─────────────────────────────┐
│      Background Layer       │
│                             │
│     Solid color: #34495E    │
│     (Wet Asphalt)           │
│                             │
└─────────────────────────────┘
```

### Required Icon Sizes

| Asset | Size | Format | Location |
|:------|:-----|:-------|:---------|
| Play Store Icon | 512×512 | PNG (32-bit, no alpha) | Play Console upload |
| Foreground (xxxhdpi) | 432×432 | WebP or PNG | `res/mipmap-xxxhdpi/ic_launcher_foreground.webp` |
| Foreground (xxhdpi) | 324×324 | WebP or PNG | `res/mipmap-xxhdpi/ic_launcher_foreground.webp` |
| Foreground (xhdpi) | 216×216 | WebP or PNG | `res/mipmap-xhdpi/ic_launcher_foreground.webp` |
| Foreground (hdpi) | 162×162 | WebP or PNG | `res/mipmap-hdpi/ic_launcher_foreground.webp` |
| Foreground (mdpi) | 108×108 | WebP or PNG | `res/mipmap-mdpi/ic_launcher_foreground.webp` |

### Adaptive Icon XML
```xml
<!-- res/mipmap-anydpi-v26/ic_launcher.xml -->
<adaptive-icon xmlns:android="http://schemas.android.com/apk/res/android">
    <background android:drawable="@color/ic_launcher_background" />
    <foreground android:drawable="@mipmap/ic_launcher_foreground" />
    <monochrome android:drawable="@mipmap/ic_launcher_foreground" />
</adaptive-icon>
```

### Icon Testing Checklist
*   [ ] Looks distinct at 48×48dp (launcher grid)
*   [ ] Readable on circle (Pixel), squircle (Samsung), rounded square (Xiaomi) shapes
*   [ ] Monochrome version works for Android 13+ themed icons
*   [ ] No text in the icon (doesn't scale well)
*   [ ] Foreground fits within the 66dp safe zone (on 108dp canvas)

## 2. Play Store Feature Graphic

| Spec | Value |
|:-----|:------|
| Dimensions | 1024 × 500 px |
| Format | PNG or JPEG |
| Max File Size | 1 MB |

### Design Brief
```
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│   [App Icon]   BD Varsity CGPA Calculator                    │
│                                                              │
│   "Calculate CGPA for 17+ BD Universities"                   │
│                                                              │
│   DU · NU · BUET · BRAC · NSU · AIUB · IUB · ...           │
│                                                              │
│   Background: Gradient from #34495E to #2ECC71               │
└──────────────────────────────────────────────────────────────┘
```

*   **Background**: Dark-to-green gradient (left to right)
*   **Text**: White, Outfit font, clean and minimal
*   **Icon**: App icon placed on the left
*   **University names**: Subtle chips or text list showing key university abbreviations
*   **No device frames**: Feature graphic should not contain phone mockups

*   **Minimum**: 4 screenshots required
*   **Maximum**: 8 screenshots allowed
*   **Content**: Must show the actual running application — no concept mockups
*   **Status bar**: Clean status bar (full battery, WiFi, reasonable time)
*   **Demo data**: Use realistic course names and grades (not "Test Course 1")
*   **No watermarks**: No developer branding overlaid on screenshots
*   **Orientation**: Portrait only
