## 1. Table: `clothing_items`
Stores logged clothing item records.
```
| Column Name | SQLite Data Type | Constraints | Description |
| :--- | :--- | :--- | :--- |
| `id` | `INTEGER` | PRIMARY KEY AUTOINCREMENT | Unique item ID |
| `image_path` | `TEXT` | NOT NULL | Local sandbox absolute file path |
| `category` | `TEXT` | NOT NULL | tops, bottoms, shoes, outer |
| `color_hex` | `TEXT` | NOT NULL | Custom color identifier |
| `created_at` | `INTEGER` | NOT NULL | Creation log timestamp |
```

## 2. Room Entity Kotlin Implementation
```
```kotlin
package com.saviorsystems.outfitcanvas.data.database.entities
```
import androidx.room.ColumnInfo
import androidx.room.Entity
import androidx.room.PrimaryKey
```
@Entity(tableName = "clothing_items")
data class ClothingItemEntity(
    @PrimaryKey(autoGenerate = true) val id: Long = 0L,
    @ColumnInfo(name = "image_path") val imagePath: String,
    val category: String,
    @ColumnInfo(name = "color_hex") val colorHex: String,
    @ColumnInfo(name = "created_at") val createdAt: Long = System.currentTimeMillis()
)
