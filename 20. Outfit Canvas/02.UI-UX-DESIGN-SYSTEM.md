# 02. UI/UX Design System
`
This document outlines structural visual tokens for **Outfit Canvas**.
`
---
`
## 1. Color Palette
The primary style uses **Warm Clay** (`#D84315`) to evoke earthiness, fashion styling, and terracotta warmth.
`
### Dark Theme (Default)
*   **Background**: `#121212` / `#221614` (Deep Terracotta Charcoal)
*   **Primary (Brand)**: `#D84315` (Warm Clay)
*   **Secondary (Brand Accent)**: `#FF8A65` (Terracotta/Peach)
*   **On-Background (Text Primary)**: `#FBE9E7` (Clay Ice White)
*   **On-Background (Text Secondary)**: `#FFAB91` (Muted Peach)
`
### Light Theme
*   **Background**: `#FAFAFA`
*   **Primary (Brand)**: `#BF360C`
*   **Secondary (Brand Accent)**: `#FF5722`
*   **On-Background (Text Primary)**: `#2E0F05`
*   **On-Background (Text Secondary)**: `#FF7043`
`
---
`
## 2. Typography
*   **Display Title**: `Outfit Bold`, 26sp
*   **Header / Subheadings**: `Outfit Medium`, 18sp
*   **Labels**: `Inter Regular`, 14sp
