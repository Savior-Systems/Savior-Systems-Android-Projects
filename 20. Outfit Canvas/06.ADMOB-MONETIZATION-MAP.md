# 06. AdMob Monetization Map

This document outlines AdMob integrations for **Outfit Canvas**.

---

## 1. Placements
*   **Adaptive Banner**: Docked at the bottom of the Closet Grid view.
*   **Interstitial Ad**: Triggered when the user saves an outfit combination to the calendar or exports a styled collage.
*   **Zero-Ad Focus Zones**: The styling Canvas interface, zoomable cards, and color matching swatches must remain strictly ad-free to keep the wardrobe styling flow smooth.

---

## 2. 180s Cooldown Guard
A global cooldown is enforced to prevent ad spam.

```kotlin
package com.saviorsystems.outfitcanvas.utils

object AdMobHelper {
    private var lastAdShowTime: Long = 0L
    private const val COOLDOWN_MS = 180_000L

    fun canShowInterstitial(): Boolean = (System.currentTimeMillis() - lastAdShowTime) >= COOLDOWN_MS
    fun markAdShown() { lastAdShowTime = System.currentTimeMillis() }
}
```
