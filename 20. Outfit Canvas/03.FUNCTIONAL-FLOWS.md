# 03. Functional Flows

This document details interactive sequences for **Outfit Canvas**.

---

## 1. Wardrobe Item Snapping Flow
```mermaid
sequenceDiagram
    participant User
    participant Camera
    participant SandboxStorage
    participant RoomDB

    User->>Camera: Snap photo of a leather jacket
    Camera->>SandboxStorage: Write compressed jpeg to app private directory
    SandboxStorage->>RoomDB: Insert ClothingItemEntity(path="private/jacket.jpg", cat="outerwear")
    RoomDB-->>User: Refresh Closet grid showing new jacket thumbnail
```

---

## 2. Outfit Canvas Drag-and-Match Flow
```mermaid
graph TD
    A[Launch Outfit Builder Canvas] --> B[Load categories cards list at bottom]
    B -->|User drags item up| C[Render dynamic card on visualizer canvas]
    C -->|Coordinate tops and bottoms| D[Save configuration metadata]
    D -->|User clicks Export collage| E[Check 180s Cooldown]
    E -->|Passed| F[Show Interstitial Ad]
    E -->|Active| G[Save to Local Gallery directly]
```
