# 04. Technical Architecture

## Tech Stack
- **Language**: Kotlin 1.9+
- **UI Toolkit**: Jetpack Compose (Material 3)
- **Architecture**: MVI (Model-View-Intent)
- **Dependency Injection**: Hilt
- **Persistence**: Room DB (Weight History)

## Build & Module Setup
Single-module architecture (`app` module) utilizing `buildSrc` or `libs.versions.toml` for standardized dependency management across Savior Systems apps.

## Compose Navigation
Using `androidx.navigation.compose` with a centralized `NavGraph`.
