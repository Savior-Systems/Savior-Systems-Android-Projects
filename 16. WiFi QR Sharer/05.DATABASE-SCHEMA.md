# 05. Database Schema

This document details storage rules for **WiFi QR Sharer**.

---

## 1. Storage Choice
The application stores generated profiles offline using **Jetpack DataStore** as JSON lists, avoiding database locking overhead.

---

## 2. Preference Schema
The saved networks list is mapped to a JSON array under the preference key:
*   `saved_networks_list`: `STRING` (Serialized JSON array of WifiNetwork items)

### JSON Schema per Network Item
```json
{
  "ssid": "SSID_NAME",
  "password": "PLAINTEXT_PASSWORD",
  "encryption_type": "WPA",
  "is_hidden": false,
  "timestamp": 1780817006860
}
```
