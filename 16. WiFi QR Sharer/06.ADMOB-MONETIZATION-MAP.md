## 1. Placements
*   **Adaptive Banner**: Anchored at the bottom of the input generation form.
*   **Interstitial Ad**: Triggered when saving network configuration properties to history.
*   **Zero-Ad Focus Zones**: Live Camera Scanning finder screens and fullscreen QR code displays are kept 100% ad-free so guest sharing remains smooth.

## 2. 180s Cooldown Guard
A global cooldown is enforced to prevent ad spam.
```
```kotlin
package com.saviorsystems.wifiqrsharer.utils
```
object AdMobHelper {
    private var lastAdShowTime: Long = 0L
    private const val COOLDOWN_MS = 180_000L
```
    fun canShowInterstitial(): Boolean = (System.currentTimeMillis() - lastAdShowTime) >= COOLDOWN_MS
    fun markAdShown() { lastAdShowTime = System.currentTimeMillis() }
}
