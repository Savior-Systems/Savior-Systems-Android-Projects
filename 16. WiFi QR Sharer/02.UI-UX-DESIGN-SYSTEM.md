# 02. UI/UX Design System
```
This document outlines structural visual tokens for **WiFi QR Sharer**.
```
---
```
## 1. Color Palette
The primary style uses **Electric Amber** (`#FF8F00`) to represent signal strength, accessibility, and high performance.
```
### Dark Theme (Default)
*   **Background**: `#121212` / `#221C14` (Warm Charcoal)
*   **Primary (Brand)**: `#FF8F00` (Electric Amber)
*   **Secondary (Brand Accent)**: `#00B0FF` (Sky Blue)
*   **On-Background (Text Primary)**: `#FFEEDD` (Soft Cream White)
*   **On-Background (Text Secondary)**: `#FFCC80` (Muted Orange Gray)
```
### Light Theme
*   **Background**: `#FAFAFA`
*   **Primary (Brand)**: `#E65100` (Deep Amber Red)
*   **Secondary (Brand Accent)**: `#0091EA`
*   **On-Background (Text Primary)**: `#2E1C0C`
*   **On-Background (Text Secondary)**: `#F57C00`
```
---
```
## 2. Typography
*   **Display Title**: `Outfit Bold`, 26sp
*   **Header / Subheadings**: `Outfit Medium`, 18sp
*   **Inputs & Text**: `Inter Regular`, 14sp
```
---
```
## 3. Gestures
*   **Zoom QR**: Pinch-to-zoom gesture on the generated QR graphic to expand it full-screen for easier guest scanning.
```
