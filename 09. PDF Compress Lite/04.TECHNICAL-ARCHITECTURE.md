## 2. Core Compression Engine (Kotlin)
```
This engine decodes PDF pages into Bitmaps using native `PdfRenderer`, applies standard compression properties via JPEG stream conversion, and writes them back into a new PDF using `PdfDocument` - completely offline.
```
```kotlin
package com.saviorsystems.pdfcompressor.engine
```
import android.content.Context
import android.graphics.Bitmap
import android.graphics.pdf.PdfDocument
import android.graphics.pdf.PdfRenderer
import android.net.Uri
import android.os.ParcelFileDescriptor
import java.io.ByteArrayOutputStream
import java.io.File
import java.io.FileOutputStream
```
object PdfCompressEngine {
```
    fun compressPdf(
        context: Context,
        inputUri: Uri,
        outputFile: File,
        jpegQuality: Int, // 1 to 100
        onProgress: (Int, Int) -> Unit // currentPage, totalPages
    ): Boolean {
        var pfd: ParcelFileDescriptor? = null
        var renderer: PdfRenderer? = null
        val outputPdf = PdfDocument()
```
        try {
            pfd = context.contentResolver.openFileDescriptor(inputUri, "r") ?: return false
            renderer = PdfRenderer(pfd)
            val pageCount = renderer.pageCount
```
            for (i in 0 until pageCount) {
                val page = renderer.openPage(i)
```
                // Construct bitmap matching target dimensions
                val bitmap = Bitmap.createBitmap(page.width, page.height, Bitmap.Config.ARGB_8888)
                page.render(bitmap, null, null, PdfRenderer.Page.RENDER_MODE_FOR_DISPLAY)
```
                // Compress bitmap using JPEG format
                val byteStream = ByteArrayOutputStream()
                bitmap.compress(Bitmap.CompressFormat.JPEG, jpegQuality, byteStream)
```
                // Write compressed page elements back to output PDF
                val pdfPageInfo = PdfDocument.PageInfo.Builder(page.width, page.height, i).create()
                val pdfPage = outputPdf.startPage(pdfPageInfo)
```
                // Draw re-compressed bitmap onto PDF page canvas
                pdfPage.canvas.drawBitmap(bitmap, 0f, 0f, null)
                outputPdf.finishPage(pdfPage)
```
                page.close()
                bitmap.recycle()
```
                onProgress(i + 1, pageCount)
            }
```
            // Flush changes to output file destination
            FileOutputStream(outputFile).use { out ->
                outputPdf.writeTo(out)
            }
            return true
        } catch (e: Exception) {
            e.printStackTrace()
            return false
        } finally {
            outputPdf.close()
            renderer?.close()
            pfd?.close()
        }
    }
}

## 3. Compression presets mapping
*   **Low (High Quality)**: JPEG Quality = 85%, renders at 1.0x native dimensions.
*   **Medium (Balanced)**: JPEG Quality = 70%, renders at 0.8x native dimensions.
*   **High (Max Compression)**: JPEG Quality = 50%, renders at 0.6x native dimensions.
