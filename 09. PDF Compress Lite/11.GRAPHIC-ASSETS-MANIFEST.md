## 2. Adaptive Icon Layout Configuration

Designed to match Material 3 specifications:
*   **Background Layer (`res/drawable/ic_launcher_background.xml`)**: Solid hex `#C62828` (Crimson Red) background.
*   **Foreground Layer (`res/drawable/ic_launcher_foreground.xml`)**: Vector path showing a document sheet being squeezed or compressed by arrows in white and light pink accents.
*   **Monochrome Layer (`res/drawable/ic_launcher_monochrome.xml`)**: Minimalist vector rendering supporting theme-based coloring.
