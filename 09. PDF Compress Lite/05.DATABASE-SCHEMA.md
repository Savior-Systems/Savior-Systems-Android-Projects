Tracking file statistics and size optimizations history locally on the SQLite container.

```
## 1. Entity: `compression_history`
```
This table persists the logs of compression jobs run by the user.
```
| Column Name | SQLite Data Type | Constraints / Keys | Description |
| :--- | :--- | :--- | :--- |
| `id` | INTEGER | PRIMARY KEY AUTOINCREMENT | Primary Key |
| `file_name` | TEXT | NOT NULL | Name of the compressed document |
| `original_size` | INTEGER | NOT NULL | Original size in bytes |
| `compressed_size` | INTEGER | NOT NULL | Output size in bytes |
| `percentage_saved` | REAL | NOT NULL | Computed savings ratio |
| `created_at` | INTEGER | NOT NULL | Execution epoch timestamp |
```

## 2. Room DAO Interface
```
```kotlin
package com.saviorsystems.pdfcompressor.data.database
```
import androidx.room.*
import kotlinx.coroutines.flow.Flow
```
@Dao
interface HistoryDao {
```
    @Query("SELECT * FROM compression_history ORDER BY created_at DESC")
    fun getAllHistoryFlow(): Flow<List<HistoryEntity>>
```
    @Insert(onConflict = OnConflictStrategy.REPLACE)
    suspend fun insertLog(log: HistoryEntity): Long
```
    @Query("DELETE FROM compression_history")
    suspend fun clearAllLogs()
```
    @Query("SELECT SUM(original_size - compressed_size) FROM compression_history")
    fun getTotalBytesSavedFlow(): Flow<Long?>
}
