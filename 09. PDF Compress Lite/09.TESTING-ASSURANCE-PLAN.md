## 1. Automated Test Scope

```
---
```
## 2. Manual QA Matrix (Edge Cases)
```
| QA Case | Action | Expected Behavior |
| :--- | :--- | :--- |
| **Password Protected File** | Select a locked PDF via picker. | The app blocks compression steps and triggers a warning message: *"This document is password protected."* |
| **Already Optimized Document** | Compress a highly optimized file on "High" preset. | Verify the output is not saved if it results in an increased file size. |
| **Multi-page Scanned Document** | Import a 50-page scanned document. | Verify compression completes without running out of memory (OOM) by recycling bitmaps during iteration. |
| **URI Revocation** | Revoke storage authorization during compression. | Abort the compression process cleanly and notify the user. |
```
---
```
## 3. Performance & Compression Benchmarks
*   **Cold Boot Time**: `< 400ms`.
*   **Active Heap Memory**: Under `96MB` during multi-page rendering loops.
*   **Compression Speed**: `< 0.5s` per page on mid-range devices.
