# 01. Product Requirement Document (PRD) — Prayer Time Helper
```
## 1. Vision & Strategy
*   **App Name**: Prayer Time Helper
*   **Package**: `com.saviorsystems.prayertimehelper`
*   **Category**: Lifestyle / Religion
*   **Core Objective**: Build a highly accurate, completely offline prayer time calculator and Qibla compass tailored specifically for the South Asian Muslim community (Bangladesh, India, Pakistan).
*   **Privacy Promise**: Zero background location tracking. No data collection. The app calculates times locally on the device based on a one-time foreground location fetch or manual city selection.
*   **Target Keywords**: `prayer time`, `namaz time bangladesh`, `qibla finder`, `azan alarm`, `salat times`, `offline namaz app`
```

## 2. Target Persona
```
### 2.1 Primary — The Devout Professional (South Asia)
*   **Demographics**: Age 18–50, based in Bangladesh, Pakistan, or India.
*   **Pain Points**: Irritated by bloated Islamic apps filled with intrusive ads, background tracking, and battery drain. Needs accurate Hanafi Asr times.
*   **Goal**: Wants a clean, reliable app that alerts them precisely at Azan time and works without an internet connection.
```
### 2.2 Secondary — The Traveling Worshipper
*   **Pain Points**: Needs to find the Qibla direction quickly when in a new hotel or unfamiliar location, often without reliable Wi-Fi.
*   **Goal**: Open the app, view an offline compass, and pray.
```

## 3. Core Feature List
```
### 3.1 On-Device Calculation Engine (Offline First)
*   **Technology**: Uses the `batoulapps/adhan-kotlin` library for astronomical calculations.
*   **Methods**: Supports multiple conventions, defaulting to *University of Islamic Sciences, Karachi* for the South Asian market.
*   **Asr Calculation**: Explicit toggle for *Hanafi* (later shadow) vs. *Shafi'i/Maliki/Hanbali* (standard shadow).
*   **No API Dependency**: Zero network calls required for daily timings.
```
### 3.2 Dynamic Prayer Dashboard
*   **Current Prayer Highlight**: Visual indicator of the current prayer and a live countdown timer to the next prayer.
*   **Daily Timings**: Displays Fajr, Sunrise, Dhuhr, Asr, Maghrib, and Isha.
*   **Hijri Date Display**: Converts Gregorian date to Islamic Hijri calendar natively.
```
### 3.3 Qibla Compass
*   **Sensor Integration**: Uses device Magnetometer and Accelerometer to render a smooth 2D compass.
*   **Target Calculation**: Calculates the great-circle distance and bearing from the user's coordinates to the Kaaba (21.4225° N, 39.8262° E).
*   **Haptic Feedback**: Phone vibrates subtly when exactly aligned with the Qibla.
```
### 3.4 Azan Notifications & Alarms
*   **Local Alarms**: Uses Android `AlarmManager` to trigger exact notifications.
*   **Customization**: Users can toggle notifications per prayer (e.g., Silent for Dhuhr, Full Azan audio for Fajr).
*   **Pre-Alerts**: Optional reminders (e.g., "15 minutes until Maghrib").
```
### 3.5 Location Management (Privacy Safe)
*   **Foreground Location**: Fetches GPS coordinates only once while the app is open.
*   **Manual Selection**: Users can type their city name and fetch coordinates via a one-time Geocoding lookup, completely bypassing GPS permissions if desired.
```
### 3.6 Bonus Utility: Tasbih Counter
*   **Digital Counter**: Simple tap-to-count screen for Dhikr.
*   **Haptic Taps**: Vibrates on every 33rd tap.
```
