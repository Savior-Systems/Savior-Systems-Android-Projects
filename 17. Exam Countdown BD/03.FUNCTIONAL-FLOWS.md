# 03. Functional Flows

This document details interactive maps for **Exam Countdown BD**.

---

## 1. Exam Addition Flow
```mermaid
sequenceDiagram
    participant User
    participant Dashboard
    participant InputForm
    participant RoomDB

    User->>Dashboard: Tap "Add Exam" button
    Dashboard->>InputForm: Slide up fields (Title, Date, Time, Category)
    User->>InputForm: Input title="BCS Prelim", date="2026-11-20", cat="BCS"
    User->>InputForm: Tap "Create Countdown"
    InputForm->>RoomDB: Insert ExamEntity
    RoomDB-->>Dashboard: Refresh list Flow
    Dashboard->>User: Renders card with running countdown clock (seconds updates)
```

---

## 2. Notification Dispatcher Flow
```mermaid
graph TD
    A[Alarm Manager Check] -->|Time remaining <= 24 hours| B[Trigger Notification Broadcast]
    B -->|Check POST_NOTIFICATIONS status| C{Granted?}
    C -- Yes --> D[Build dynamic alarm notification: Exam name + Days left]
    C -- No --> E[Log warning locally to debug system]
```
