# 09. Testing & Quality Assurance Plan

This document outlines testing strategies, unit test implementations, and manual QA checklists for **Exam Countdown BD**.

---

## 1. Automated Unit Test Cases (Time Calculator)

```kotlin
package com.saviorsystems.examcountdownbd

import com.saviorsystems.examcountdownbd.utils.CountdownMath
import org.junit.Assert.assertEquals
import org.junit.Test

class CountdownMathTest {

    @Test
    fun verifyRemainingTimeCalculatesCorrectly() {
        val now = System.currentTimeMillis()
        
        // 2 days later in milliseconds = 2 * 86400 * 1000 = 172800000 ms
        val target = now + 172800000 + 5000 // Add a 5s buffer to prevent test race conditions
        
        val remaining = CountdownMath.calculateTimeRemaining(target)
        assertEquals(2, remaining.days)
    }
}
```
