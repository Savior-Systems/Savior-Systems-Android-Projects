# 06. AdMob Monetization Map
`
This document outlines AdMob integrations for **Exam Countdown BD**.
`
---
`
## 1. Placements
*   **Adaptive Banner**: Docked at the bottom of the main dashboard.
*   **Interstitial Ad**: Triggered when a user deletes an exam timeline card or exits Settings.
*   **Zero-Ad Focus Zones**: Active countdown panels, sub-todo checklists, and deadline addition modals must remain strictly ad-free to keep study focus high.
`
---
`
## 2. 180s Cooldown Guard
A global cooldown is enforced to prevent ad spam.
`
```kotlin
package com.saviorsystems.examcountdownbd.utils
`
object AdMobHelper {
    private var lastAdShowTime: Long = 0L
    private const val COOLDOWN_MS = 180_000L
`
    fun canShowInterstitial(): Boolean = (System.currentTimeMillis() - lastAdShowTime) >= COOLDOWN_MS
    fun markAdShown() { lastAdShowTime = System.currentTimeMillis() }
}
```
