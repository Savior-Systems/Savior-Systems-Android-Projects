## 1. Table: `exams`
Stores the target exams and deadlines.
```
| Column Name | SQLite Data Type | Constraints | Description |
| :--- | :--- | :--- | :--- |
| `id` | `INTEGER` | PRIMARY KEY AUTOINCREMENT | Unique exam ID |
| `title` | `TEXT` | NOT NULL | Exam name (e.g. BCS Prelims) |
| `target_timestamp` | `INTEGER` | NOT NULL | Milliseconds target deadline |
| `category` | `TEXT` | NOT NULL | SSC, HSC, BCS, Admission |
| `created_at` | `INTEGER` | NOT NULL | Creation log timestamp |
```

## 2. Room Entity Kotlin Implementation
```
```kotlin
package com.saviorsystems.examcountdownbd.data.database.entities
```
import androidx.room.ColumnInfo
import androidx.room.Entity
import androidx.room.PrimaryKey
```
@Entity(tableName = "exams")
data class ExamEntity(
    @PrimaryKey(autoGenerate = true) val id: Long = 0L,
    val title: String,
    @ColumnInfo(name = "target_timestamp") val targetTimestamp: Long,
    val category: String,
    @ColumnInfo(name = "created_at") val createdAt: Long = System.currentTimeMillis()
)
