# 06. AdMob Monetization Map

This document outlines AdMob integrations for **Daily Quotes Maker**.

---

## 1. Placements
*   **Adaptive Banner**: Docked at the bottom of the Quote Explorer page.
*   **Interstitial Ad**: Triggered when a user successfully exports / saves a quote poster graphic to the gallery.
*   **Zero-Ad Focus Zones**: The styling Canvas, font lists, and backdrop selectors must remain strictly ad-free to keep the creative editing workspace clean.

---

## 2. 180s Cooldown Guard
A global cooldown is enforced to prevent ad spam.

```kotlin
package com.saviorsystems.dailyquotesmaker.utils

object AdMobHelper {
    private var lastAdShowTime: Long = 0L
    private const val COOLDOWN_MS = 180_000L

    fun canShowInterstitial(): Boolean = (System.currentTimeMillis() - lastAdShowTime) >= COOLDOWN_MS
    fun markAdShown() { lastAdShowTime = System.currentTimeMillis() }
}
```
