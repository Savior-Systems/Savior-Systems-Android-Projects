# 12. Logging & Analytics

This document details logging rules and analytics setup for **Daily Quotes Maker**.

---

## 1. Privacy Mandate
*   **Zero PII**: Input quotes, customized texts, and author fields must **never** be logged to analytics dashboards.

---

## 2. Analytics Events

### Event: Quote Poster Exported (`poster_exported`)
Fired when a user saves a poster image.
*   `backdrop_type`: `TEXT` ("solid", "gradient", "texture")
*   `font_family_selected`: `TEXT` ("montserrat", "caveat", "playfair")
