# 01. Product Requirement Document (PRD)
```
## 1. Vision & Strategy
*   **App Name**: Daily Quotes Maker
*   **Category**: Art & Design / Entertainment
*   **Core Objective**: Build a fast, offline quote visualizer and poster creator. It targets search intent for: `quote maker`, `text on photo`, and `wallpaper maker`.
*   **Value Proposition**: Local text-on-photo layout editors, offline font libraries, premium template cards, and instant sharing.
```
---
```
## 2. Target Persona
### Persona: Chloe (Aspiring Lifestyle Influencer)
*   **Needs**: Quickly design beautiful, text-styled quote cards using inspiring templates and post them directly to Instagram or WhatsApp.
*   **Pain Points**: Design apps that require cloud sign-up, watermark final outputs, or have massive load times due to heavy media assets.
*   **How the App Solves it**: Chloe opens the app, types or picks a quote, selects an offline backdrop texture or solid color, applies a Google Font style, and exports a watermark-free PNG directly from the local layout.
```
---
```
## 3. Core Feature List
1.  **Visual Styling Canvas**:
    *   Dynamic canvas allowing text resizing, alignment adjustment, line-spacing, and color picking.
    *   Preloaded background texture categories (Solid, Gradient, Abstract, Retro).
    *   Clean selector chip bar for local system and custom bundled fonts.
2.  **Quote Discovery Hub**:
    *   Offline catalog of quotes categorized by mood (Inspirational, Success, Wisdom, Friendship).
3.  **Encrypted Favorites Deck**:
    *   Local bookmarking to store favorite quotes inside a Room database table.
```
---
```
## 4. Monetization & Ad Zones
*   **Zero-Ad Zones**:
    *   The Quote Editor canvas workspace and background styling selectors are 100% ad-free to keep the user's creative workflow uninterrupted.
*   **Monetization**:
    *   Banner ad at the bottom of the quote explorer page.
    *   Interstitial ad on exporting/saving the high-resolution PNG to the gallery, governed by a **180s cooldown**.
```
