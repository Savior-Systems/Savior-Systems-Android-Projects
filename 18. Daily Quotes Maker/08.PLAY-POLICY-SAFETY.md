# 08. Play Store Policy & Data Safety

This document outlines Google Play privacy policies for **Daily Quotes Maker**.

---

## 1. Requested Permissions
To maximize user trust and prevent automated app flags during review, the app requests the following permission on legacy Android devices:

| Permission | API Level | Purpose | Necessity |
| :--- | :--- | :--- | :--- |
| `android.permission.WRITE_EXTERNAL_STORAGE` | API < 29 | Save exported quote images to local public gallery. | **Legacy Required** only on Android 9 or lower. |

No location, contact, phone, camera, or network permissions are declared.

---

## 2. Google Play Data Safety Declarations

### A. Data Collection
*   **Data Collected**: None.
*   The application does not collect or transmit any user data (e.g., custom quotes, styled graphics) to external servers.

### B. Data Sharing
*   **Data Shared**: None.
*   No personal data, device identifiers, or quote creations are shared with third-party networks or service providers.

### C. Data Security Measures
*   **Encryption**: No data leaves the device.
*   **Data Deletion**: Users can delete bookmarked quotes by clearing app storage in system settings.
