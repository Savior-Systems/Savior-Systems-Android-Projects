## 1. Table: `favorite_quotes`
Stores bookmarked quotes.
```
| Column Name | SQLite Data Type | Constraints | Description |
| :--- | :--- | :--- | :--- |
| `id` | `INTEGER` | PRIMARY KEY AUTOINCREMENT | Unique quote ID |
| `text` | `TEXT` | NOT NULL | Body of the quote |
| `author` | `TEXT` | NOT NULL | Quote author |
| `timestamp` | `INTEGER` | NOT NULL | Date bookmarked |
```

## 2. Room Entity Kotlin Implementation
```
```kotlin
package com.saviorsystems.dailyquotesmaker.data.database.entities
```
import androidx.room.Entity
import androidx.room.PrimaryKey
```
@Entity(tableName = "favorite_quotes")
data class QuoteEntity(
    @PrimaryKey(autoGenerate = true) val id: Long = 0L,
    val text: String,
    val author: String,
    val timestamp: Long = System.currentTimeMillis()
)
```

## 3. Database DAO Queries
```
```kotlin
package com.saviorsystems.dailyquotesmaker.data.database
```
import androidx.room.*
import com.saviorsystems.dailyquotesmaker.data.database.entities.QuoteEntity
import kotlinx.coroutines.flow.Flow
```
@Dao
interface QuoteDao {
    @Query("SELECT * FROM favorite_quotes ORDER BY timestamp DESC")
    fun getFavoriteQuotes(): Flow<List<QuoteEntity>>
```
    @Insert(onConflict = OnConflictStrategy.REPLACE)
    suspend fun insertFavorite(quote: QuoteEntity): Long
```
    @Delete
    suspend fun deleteFavorite(quote: QuoteEntity)
}
