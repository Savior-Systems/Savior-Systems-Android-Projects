# 02. UI/UX Design System

This document details visual guidelines for **Daily Quotes Maker**.

---

## 1. Color Palette
The primary style uses **Soft Rose Pink** (`#D81B60`) to evoke creativity, aesthetics, and modern styling.

### Dark Theme (Default)
*   **Background**: `#121212` / `#1F1417` (Deep Rose Charcoal)
*   **Primary (Brand)**: `#D81B60` (Soft Rose Pink)
*   **Secondary (Brand Accent)**: `#F06292` (Light Pink)
*   **On-Background (Text Primary)**: `#FCE4EC` (Rose Ice White)
*   **On-Background (Text Secondary)**: `#F48FB1` (Muted Rose)

### Light Theme
*   **Background**: `#FAFAFA`
*   **Primary (Brand)**: `#C2185B`
*   **Secondary (Brand Accent)**: `#E91E63`
*   **On-Background (Text Primary)**: `#2C0D17`
*   **On-Background (Text Secondary)**: `#EC407A`

---

## 2. Typography
*   **Display Title**: `Outfit Bold`, 26sp
*   **Custom Quote Styles (Canvas)**: Uses downloadable/bundled fonts (e.g. `Playfair Display` for serif elegance, `Montserrat` for modern bold, `Caveat` for handwritten accents).
