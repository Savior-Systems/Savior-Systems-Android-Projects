# 09. Testing & Quality Assurance Plan

This document outlines testing strategies, unit test implementations, and manual QA checklists for **Daily Quotes Maker**.

---

## 1. Automated Unit Test Cases (Text Checks)

```kotlin
package com.saviorsystems.dailyquotesmaker

import com.saviorsystems.dailyquotesmaker.data.database.entities.QuoteEntity
import org.junit.Assert.assertTrue
import org.junit.Test

class QuoteValidationTest {

    @Test
    fun verifyBlankQuoteRejection() {
        val quote = QuoteEntity(text = "Hello", author = "Anon")
        
        // Quote body must not be blank
        assertTrue(quote.text.isNotBlank())
        assertTrue(quote.author.isNotBlank())
    }
}
```
