# 04. Technical Architecture Blueprint

## 1. Architecture Choice: MVVM (Model-View-ViewModel)
We will use modern Android development standards. The Rookie Developer MUST strictly separate concerns.

- **UI Layer:** Jetpack Compose (or XML if team prefers, but Compose is recommended). Observes StateFlows from ViewModel.
- **Domain/ViewModel Layer:** Connects UI to Data. Holds UI state. Uses Kotlin Coroutines for async operations.
- **Data Layer:** Room Database (SQLite) for local storage. Repository pattern to abstract the DB logic.

## 2. Recommended Tech Stack
- **Language:** Kotlin
- **UI:** Jetpack Compose (Material 3)
- **Local Database:** Room Database
- **Dependency Injection:** Hilt / Dagger
- **Asynchrony:** Kotlin Coroutines & Flow
- **Navigation:** Jetpack Navigation Component
