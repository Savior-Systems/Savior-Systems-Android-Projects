# 02. UI/UX Design System

## 1. Design Philosophy
The app should feel modern, snappy, and visually appealing. We will use **Material Design 3 (Material You)** principles. The UI must feel premium, utilizing ample whitespace and smooth rounded corners.

## 2. Color Palette Strategy
We want vibrant, aesthetic colors that appeal to a young demographic.
- **Primary Color:** Deep Purple (`#6200EA`)
- **Secondary / Accent:** Amber Accent (`#FFAB00`)
- **Background (Light Mode):** Clean White (`#FFFFFF`) or very light gray (`#FAFAFA`)
- **Background (Dark Mode):** Pure Black (`#000000`) for AMOLED screens or deep gray (`#121212`)
- **Card Surface:** Slightly elevated colors from the background (e.g., `#F5F5F5` in light, `#1E1E1E` in dark mode).

## 3. Typography
Use modern Google Fonts to elevate the premium feel.
- **Headings/Titles:** `Outfit` (Bold & Semi-Bold)
- **Body Text (The Captions):** `Inter` or standard `Roboto` (Medium 16sp, line height 24sp for readability).

## 4. Component Guidelines (For Developers)
- **Buttons:** Use Material 3 FilledTonal buttons for primary actions. Use IconButtons for "Copy" and "Share".
- **Cards:** All items in the list should be inside `Card` components with an 8dp or 12dp `RoundedCornerShape`. Elevation should be subtle (2dp to 4dp).
- **Icons:** Use standard Material Icons (`ContentCopy`, `Share`, `Favorite`, `FavoriteBorder`).

## 5. UI States to Handle
- **Empty State:** If the "Favorites" screen is empty, show an illustration or a text saying "No favorites yet. Tap the heart icon to save some!"
- **Loading State:** Not heavily needed since it's an offline DB, but use a subtle `CircularProgressIndicator` if database initialization takes a few milliseconds on first launch.
