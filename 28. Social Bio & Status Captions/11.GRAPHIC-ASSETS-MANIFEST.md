# 11. Graphic Asset Manifest

## 1. Required Assets from Design Team
The developer will need these files before publishing:

1. **App Icon (`ic_launcher`)**
   - Dimensions: 512x512 png (for Play Store)
   - Dimensions: 108x108 vector (for Android Adaptive Icons)
   - Style: Minimalist icon, perhaps a stylized quote mark (") or a glowing profile icon.

2. **Category Icons (Vectors / XML)**
   - Developer should use Material Icons provided by Google to save time for MVP.
   - `ic_attitude` -> `Icons.Default.Whatshot` (Fire)
   - `ic_love` -> `Icons.Default.Favorite` (Heart)
   - `ic_motivation` -> `Icons.Default.Star` (Star)
   - `ic_funny` -> `Icons.Default.Face` (Smiley)
   - `ic_sad` -> `Icons.Default.WaterDrop` (Tear)

3. **Splash Screen Logo**
   - Simple transparent PNG of the app icon to use in the Android 12+ Splash Screen API.
