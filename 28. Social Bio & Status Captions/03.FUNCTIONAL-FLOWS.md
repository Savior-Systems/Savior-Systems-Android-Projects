# 03. Functional Flows & Navigation

## 1. Overall App Flowchart
This is the journey the user takes through the app. Rookie developers must wire the navigation exactly like this.

```text
[ App Launch ]
      │
      ▼
[ Splash Screen ] (Optional, Android 12+ native splash is fine)
      │
      ▼
[ Home Screen ] <──────────────┐
  - Displays Category Grid     │
  - Bottom/Top Nav for Favs    │
      │                        │
      ├─> (Taps "Favorites") ──┴─> [ Favorites Screen ]
      │                              - Displays saved captions
      ▼                              - Can unlike (removes from list)
[ Category List Screen ]
  - Displays Captions for X Category
  - User can scroll through the list
      │
      ├─> (Taps "Copy") ──> Copies text to clipboard -> Shows Snackbar "Copied!"
      │
      ├─> (Taps "Share") ─> Opens Android Share Sheet -> Select App
      │
      └─> (Taps "Heart") ─> Saves to Favorites -> Heart icon turns solid Red
```

## 2. Step-by-Step Flow Instructions for Dev
1. **App opens -> Main Activity.**
2. Initialize the Navigation Graph (`NavHost` in Compose or `NavHostFragment` in XML).
3. The starting destination is the **Home Screen**.
4. Pass the `categoryId` or `categoryName` as an argument when navigating from Home to the Category List Screen.
