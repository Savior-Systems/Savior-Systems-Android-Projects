## 3. Strict Interstitial Cooldown
`
To avoid interrupting user workflows:
```kotlin
object HabitAdManager {
    private const val AD_COOLDOWN_MS = 180_000L // 3 minutes
    private var lastAdShowTime = 0L
`
    fun canShowInterstitial(): Boolean {
        return (System.currentTimeMillis() - lastAdShowTime) >= AD_COOLDOWN_MS
    }
`
    fun recordAdShown() {
        lastAdShowTime = System.currentTimeMillis()
    }
}
```
*   **Behavior**: When a habit is checked off, if `canShowInterstitial()` is true, show the ad; otherwise, complete checkoff animation and return immediately.
