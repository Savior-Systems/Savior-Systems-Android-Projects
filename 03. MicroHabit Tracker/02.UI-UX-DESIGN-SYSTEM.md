# 02. UI/UX Design System — MicroHabit Tracker
`
This design system ensures a premium, minimalist, and responsive user interface mapped to the Health & Fitness category.
`
---
`
## 1. Color Palette (Material 3 Dynamic Mappings)
`
### Light Mode Color Scheme
*   **Primary (Brand Color):** `#3498DB` (Deep Sky Blue - represent health and clarity)
*   **Secondary:** `#2C3E50` (Slate Navy - represent stability)
*   **Background:** `#F5F7FA` (Slate Tint Soft Gray)
*   **Surface:** `#FFFFFF`
`
### Dark Mode Color Scheme
*   **Primary (Brand Color):** `#85C1E9` (Neon Sky Blue)
*   **Secondary:** `#AEB6BF` (Muted Slate)
*   **Background:** `#1C2833` (Slate Dark Gray)
*   **Surface:** `#2C3E50`
`
---
`
## 2. Typography Rules
*   **Primary Font Family:** `Outfit` (Modern, rounded aesthetic, loaded via Google Fonts).
*   **Alternative/Fallback:** `System Default Sans-Serif`.
*   **Size Mappings:**
    *   `Title Large`: 22sp (Semi-Bold) for habit name titles.
    *   `Body Medium`: 16sp (Regular) for description and dates.
    *   `Label Small`: 12sp (Medium) for streak logs and labels.
`
---
`
## 3. Gesture Animations & Micro-Interactions
*   **Swipe to Log:** Swipe-right on a habit card triggers a physics-based spring animation.
*   **Completion Burst:** A particle explosion effect (minimal, lightweight Canvas draw) triggers when the daily goal is fully completed.
*   **Dark Theme Transition:** Seamless Material 3 theme crossfade.
`
