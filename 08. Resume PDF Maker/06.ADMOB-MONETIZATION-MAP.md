# 06. AdMob Monetization Map — Resume PDF Maker

## 1. Monetization Strategy Overview
To offer a high-fidelity utility app for free without forcing paywalls, this app utilizes a hybrid model:
1. **AdMob-First Revenue:** Banners, Interstitials, and Native Ads.
2. **Rewarded Premium Unlocks:** Users watch a single rewarded video ad to unlock premium PDF templates (Classic Navy, Emerald Corporate, Onyx Modern themes) and margin/font scaling controls for **24 hours**.

---

## 2. Ad Placements Index

| Ad Format | Trigger Location / Screen | Behavior & Logic Rules |
|:---|:---|:---|
| **Adaptive Sticky Banner** | Dashboard Screen (Bottom) | Loaded on launch. Anchored at the bottom of the screen. |
| **Unified Native Ad** | Dashboard Resume List | Inserted dynamically as a grid/list item between saved resume profiles (e.g., at position #3) to maximize list eCPMs. |
| **Interstitial Ad** | PDF Export Completion | Triggered when the user taps "Export PDF". Subject to the **180-second global cooldown**. |
| **Rewarded Video Ad** | Template Selection Screen | Triggered when a user selects a premium template (Classic Navy, Emerald, Onyx) or opens the advanced customization slider (margins/fonts) if not already unlocked. Unlocks all premium features for **24 hours**. |

---

## 3. Policy-Compliant Zero-Ad Zones
* **Form Inputs & Keyboard Active:** No banner updates or interstitials are allowed while the user is actively typing in the dynamic forms or when the virtual keyboard is open. This prevents accidental invalid clicks.
* **Print PDF Canvas Phase:** No ads are shown during the PDF compilation and print preview loading phases.

---

## 4. Code Implementation: AdManager & Cooldown

The `ResumeAdManager` handles the global state of interstitial cooldowns and rewarded tokens:

```kotlin
package com.saviorsystems.resumemaker.domain.ad

object ResumeAdManager {
    private const val INTERSTITIAL_COOLDOWN_MS = 180_000L // 3 minutes
    private var lastInterstitialShowTime = 0L
    private var premiumUnlockTimestamp = 0L

    // Interstitial Logic
    fun canShowInterstitial(): Boolean {
        return (System.currentTimeMillis() - lastInterstitialShowTime) >= INTERSTITIAL_COOLDOWN_MS
    }

    fun markInterstitialShown() {
        lastInterstitialShowTime = System.currentTimeMillis()
    }

    // Rewarded Premium Token Logic
    fun isPremiumUnlocked(): Boolean {
        return System.currentTimeMillis() < premiumUnlockTimestamp
    }

    fun grantPremiumAccess() {
        premiumUnlockTimestamp = System.currentTimeMillis() + (24 * 60 * 60 * 1000L) // 24 Hours
    }
}
```

* **Fallback behavior:** If `canShowInterstitial()` is false on PDF export, bypass the ad call immediately and open the system Share/View dialog. Never block the user experience.
