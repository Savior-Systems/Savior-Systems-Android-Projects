## 1. Ad Format Placements

## 2. Zero-Ad Zones
*   **Form Text Entry**: No overlay ads trigger while text fields are active or when the virtual keyboard is open to prevent accidental clicks.
*   **PDF System Spooler**: Zero ads load during the printing layout phase to prevent interference with system processes.

## 3. Interstitial Cooldown Strategy
A strict 180-second cooldown is enforced to prevent spamming users who perform quick edits and successive PDF exports.
```
```kotlin
object ResumeAdManager {
    private const val COOLDOWN_INTERVAL_MS = 180_000L // 3 minutes
    private var lastAdShowTime = 0L
```
    fun canShowInterstitial(): Boolean {
        return (System.currentTimeMillis() - lastAdShowTime) >= COOLDOWN_INTERVAL_MS
    }
```
    fun markAdShown() {
        lastAdShowTime = System.currentTimeMillis()
    }
}
```
*   **Fallback**: If `canShowInterstitial()` resolves to false, load the PDF print adapter directly and bypass the ad call.
