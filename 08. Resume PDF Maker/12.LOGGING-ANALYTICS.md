# 12. Logging & Analysis

## 1. Critical Events to Track (Firebase Analytics)
To understand app performance and bottlenecks, log these custom events:

1. `resume_created` (Parameter: `template_chosen`)
2. `pdf_export_success` (Parameter: `page_count`)
3. `pdf_export_failed` (Parameter: `error_reason`)
4. `ad_rewarded_watched` (Parameter: `template_unlocked`)

## 2. Crashlytics
- Crucial: Log non-fatal exceptions if `PdfDocument` throws an `IOException` during generation so we can debug storage permission errors on weird OEM devices (Xiaomi, Oppo).
