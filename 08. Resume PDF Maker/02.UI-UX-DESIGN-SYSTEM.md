# 02. UI/UX Design System — Resume PDF Maker
```
Visually tailored for a professional, executive feel using the Corporate Navy color scheme.
```
---
```
## 1. Color System (Corporate Navy)
```
### Light Theme
*   **Primary (`#1A237E`)**: Deep Corporate Navy for headers, action buttons, and active template borders.
*   **Primary Container (`#E8EAF6`)**: Soft navy highlight for form category tabs.
*   **Secondary (`#0288D1`)**: Steel Blue for links, sub-headers, and accent icons.
*   **Tertiary (`#37474F`)**: Slate Gray for secondary text.
*   **Background (`#F8F9FA`)**: Clean, bright utility background.
*   **Surface (`#FFFFFF`)**: Pure white cards for form fields.
*   **On-Background (`#1A1C1E`)**: Sharp off-black text.
```
### Dark Theme
*   **Primary (`#8C9EFF`)**: Bright indigo accent for action items.
*   **Primary Container (`#1A237E`)**: Deep navy container backgrounds.
*   **Secondary (`#29B6F6`)**: Light steel blue.
*   **Tertiary (`#90A4AE`)**: Cool slate.
*   **Background (`#121318`)**: Dark charcoal-blue.
*   **Surface (`#1A1B22`)**: Form card backing.
*   **On-Background (`#E3E2E6`)**: Light gray readable text.
```
---
```
## 2. Typography Standard
*   **Font Family**: **Inter** (via Google Fonts Integration) for a geometric, modern resume look.
*   **Title Large (Form Section)**: 18sp, Semi-Bold, tracking 0.15px.
*   **Body Medium (Inputs/Labels)**: 14sp, Regular, tracking 0.25px.
```
---
```
## 3. Micro-Animations
```
### Animation A: Form Section Expander
*   **Trigger**: Clicking a section header (e.g. "Work Experience") to expand input fields.
*   **Behavior**: Smooth spring height expansion (`tween` spec over 300ms with `FastOutSlowInEasing`) and a 180-degree rotation of the dropdown arrow icon.
```
### Animation B: Template Switcher Carousel
*   **Trigger**: Selecting a different resume template.
*   **Behavior**: A subtle scale-down and horizontal fade-in transition (`fadeIn` + `scaleIn` from 0.95 to 1.0) of the preview page layout to indicate the switch.
```
