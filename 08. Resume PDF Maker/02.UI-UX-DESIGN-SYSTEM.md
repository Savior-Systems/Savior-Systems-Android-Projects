# 02. UI/UX Design System

## 1. Design Philosophy
The app must inspire confidence. We are dealing with careers. The UI should be extremely clean, productivity-focused, and distraction-free.

## 2. Color Palette Strategy
- **Primary Color:** Deep Navy Blue (`#0F172A`) - Conveys trust and professionalism.
- **Secondary / Accent:** Electric Blue (`#3B82F6`) - For action buttons (Export, Add Section).
- **Background (Light Mode):** Crisp White (`#FFFFFF`) with Ghost White (`#F8FAFC`) surfaces.
- **Background (Dark Mode):** Slate Black (`#0B0F19`) with elevated Gray surfaces (`#1E293B`).
- **Danger:** Soft Red (`#EF4444`) for deleting sections/resumes.

## 3. Typography
- **Headings:** `Inter` (Bold, Semi-Bold) - Modern and readable.
- **Body & Forms:** `Roboto` (Medium 16sp) - Native Android feel for data entry.

## 4. Component Guidelines (For Developers)
- **Data Entry Steppers:** Use a visual horizontal stepper (e.g., Personal -> Edu -> Exp) to break down the massive form. Do not put all 50 inputs on one scrolling page.
- **Expandable Cards:** In the "Work Experience" section, show entered jobs as collapsed cards. Tapping expands them for editing.
- **Floating Action Button (FAB):** Use a prominent FAB on the Dashboard to "Create New Resume".

## 5. UX Flows
- **Auto-Save:** Forms must auto-save to Room as the user types (use `Flow` with a 500ms debounce), or have a very clear "Save" button at the bottom of each section.
