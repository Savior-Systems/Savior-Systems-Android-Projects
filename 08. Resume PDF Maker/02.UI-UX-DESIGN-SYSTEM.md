# 02. UI/UX Design System

## 1. Design Philosophy
The UI/UX must project executive-level quality and focus. Because this is a utility for career building, the layout must feel high-performance, distraction-free, and structured, matching the Material 3 design spec.

---

## 2. In-App App Color System
* **Primary Color:** Deep Slate Navy (`#0A192F`) — represents trust and security.
* **Secondary / Action Accent:** Teal Blue (`#00B4D8`) — for floating actions, secondary targets, and save confirmations.
* **Neutral Backgrounds:** Soft Gray (`#F8F9FA`) for light mode, Dark Slate (`#0B0F19`) for dark mode surfaces.
* **Status Colors:** Soft Crimson (`#D90429`) for deleting or resetting profiles.

---

## 3. PDF Layout & Spacing Tokens (Print Geometry)
PDF layout coordinates use PostScript points (`1 point = 1/72 inch`). A standard A4 page is exactly **595 x 842 points**.

### Spacing Configs:
| Configuration | Point Margin (Left/Right/Top/Bottom) | Use Case |
|:---|:---|:---|
| **Compact Margin** | `36 points` (0.5 in) | High density, fits massive experience onto a single page |
| **Normal Margin** | `54 points` (0.75 in) | Balanced, professional layout standard (Recommended) |
| **Wide Margin** | `72 points` (1.0 in) | Low density, fits student CVs or brief profiles |

---

## 4. Professional Branding Palettes (PDF Themes)
The user can select from three predefined professional PDF themes:

### A. Classic Navy (Standard Corporate)
* Primary Text: Deep Navy (`#0F1E36`)
* Accent Rules/Header Dividers: Steel Blue (`#4A6B82`)
* Body Text: Dark Slate Gray (`#2C3E50`)

### B. Emerald Corporate (Medical, Finance, Law)
* Primary Text: Dark Pine Green (`#143D28`)
* Accent Rules/Header Dividers: Moss Teal (`#2E6F40`)
* Body Text: Coal (`#1C2826`)

### C. Onyx Modern (Tech, Design, Creative)
* Primary Text: Carbon Black (`#1A1A1A`)
* Accent Rules/Header Dividers: Muted Charcoal (`#555555`)
* Body Text: Off-Black (`#333333`)

---

## 5. Typography Pairings
The PDF templates support three font family configurations:
1. **Modern Sans (ATS-Optimized):** `Inter` for headings, `Roboto` for body text. Perfect for scanning algorithms.
2. **Classic Serif (Academic/Executive):** `Playfair Display` or `Georgia` for headings, `Lora` or `Times New Roman` for body text.
3. **Tech Monospace:** `SF Mono` or `Fira Code` for headings and details. Best for technical/developer profiles.

---

## 6. Key Compose Components (Form & Preview)
* **Form Sections (Expandable Accordions):** Each profile form group (Education, Work Experience) resides in a collapsable card. Collapsed cards display a summary (e.g., *"Senior Developer at Savior Systems — 2 Years"*).
* **Progress Indicator Header:** A top banner displays completeness (`Completed Sections: 4/6`) using a smooth linear progress bar.
* **Canvas Preview Window:** Renders A4 layouts page-by-page. Outlined using a clean drop-shadow card, with zoom control sliders (50% to 150%) and standard page indicators (e.g., `Page 1 of 2`).
