# 08. Play Policy & Data Safety Compliance

## 1. Storage Access Framework (Zero-Permission Export)
To comply with Google Play Store policies and prevent automatic policy rejections regarding storage access, the application uses **Scoped Storage** and requires **zero runtime permissions** (no `READ_EXTERNAL_STORAGE` or `WRITE_EXTERNAL_STORAGE` declarations).

### Implementation Logic:
* **Export PDF / JSON:** Use the system file picker (`Intent.ACTION_CREATE_DOCUMENT`) with `MIME` types `application/pdf` or `application/json`.
  ```kotlin
  val intent = Intent(Intent.ACTION_CREATE_DOCUMENT).apply {
      addCategory(Intent.CATEGORY_OPENABLE)
      type = "application/pdf"
      putExtra(Intent.EXTRA_TITLE, "My_Resume_CV.pdf")
  }
  ```
  The OS handles the file creation in the user's chosen folder (e.g., `/Downloads`), returning a writeable content URI to the application's repository layer.
* **Import JSON Backups:** Use `Intent.ACTION_OPEN_DOCUMENT` with `MIME` type `application/json` to let the user select a backup file.

## 2. Google Play Console Data Safety Questionnaire Mapping
Since the application functions completely offline, you must declare that **no user data is collected or shared**.

| Question | Console Selection | Justification / Developer Notes |
|:---|:---|:---|
| **Data Collection & Sharing** | "No, this app does not collect or share any user personal data." | All inputs (resumes, educations, contacts, certifications) reside strictly in the local SQLite Room Database. |
| **Data Encryption in Transit** | "Not applicable" | No data is sent over the network. |
| **Data Deletion Mechanism** | "Yes, users can request that their data be deleted." | Tapping "Delete Resume" or "Reset Application" in the settings screen executes database drops to permanently delete SQLite records from disk. |

## 3. AdMob Integration & Accident Click Prevention
* **Dynamic Layout Guards:** Adaptive Banner ads must use fixed layout bounds to prevent layout shifting when the ad unit loads.
* **Rewarded Ad Disclosures:** Before launching a rewarded video ad to unlock premium configurations, display a Material 3 dialog stating: *"Watch a short video ad to unlock premium layout styles, custom margins, and professional themes for the next 24 hours."* Tapping "Cancel" must dismiss the dialog immediately with no ad call.
