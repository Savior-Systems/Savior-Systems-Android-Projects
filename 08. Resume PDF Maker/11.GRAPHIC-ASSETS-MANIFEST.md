# 11. Graphic Asset Manifest

## 1. App Icon specifications
* **Icon Name:** `ic_launcher`
* **Format & Dimensions:** Adaptive Vector Icon XML + 512x512 PNG (for Play Store submission).
* **Design Concept:** A clean, flat Material Design vector graphic showing a stylized sheet of A4 paper containing structural lines and a clean circular checkmark badge in the bottom-right corner.
* **Palette:** Background: Deep Navy (`#0A192F`); Foreground/Lines: Crisp White (`#FFFFFF`) & Teal Blue (`#00B4D8`).

---

## 2. Dynamic Vector Illustrations (XML/SVG)
To keep the APK size extremely lightweight (< 5MB), all illustrative graphics must be implemented as clean, scalable vector XML files rather than rasterized images.

* **`illust_empty_dashboard.xml` (Dashboard Empty State):**
  * **Visual:** A clean, flat illustration of a blank clipboard flanked by search magnifying glasses, styled with muted corporate gray and navy tones.
  * **Role:** Displayed on the Dashboard when the Room database contains zero resume records.
* **`illust_export_success.xml` (Export Dialog):**
  * **Visual:** A document folder icon opening with a checkbox indicator, colored in Emerald Corporate green.
  * **Role:** Displayed inside the save-completion dialog.

---

## 3. Template Grid Preview Thumbnails (WebP)
* **`thumb_classic_navy.webp` (Classic Navy Layout Thumbnail):**
  * **Resolution:** 240 x 340 pixels.
  * **Visual:** A miniature graphic rendering of a one-column resume with a navy blue name header and thin dividers.
* **`thumb_emerald_corporate.webp` (Emerald Corporate Layout Thumbnail):**
  * **Resolution:** 240 x 340 pixels.
  * **Visual:** A two-column CV template containing pine-green headers.
* **`thumb_onyx_modern.webp` (Onyx Modern Layout Thumbnail):**
  * **Resolution:** 240 x 340 pixels.
  * **Visual:** A minimalist layout utilizing bold carbon-black fonts and clean margins.

## 4. UI System Icons (Material Vector XML)
* **`ic_export_pdf.xml`:** File download icon combined with a standard document sheet.
* **`ic_backup_json.xml`:** Double-arrow cloud/disk transfer symbol.
* **`ic_margins.xml`:** Ruler/guides margin boundaries indicator.
* **`ic_premium_lock.xml`:** Padlock icon combined with a star to indicate rewarded-ad premium layouts.
