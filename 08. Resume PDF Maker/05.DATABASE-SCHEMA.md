# 05. Database Schema & Room DAO — Resume PDF Maker
```
Structured SQLite tables used to serialize personal profiles, work logs, and academic milestones locally.
```
---
```
## 1. Entity: `resume_profiles` (Master Table)
```
| Column Name | SQLite Data Type | Constraints / Keys | Description |
| :--- | :--- | :--- | :--- |
| `profile_id` | INTEGER | PRIMARY KEY AUTOINCREMENT | Primary key |
| `name` | TEXT | NOT NULL | Full Name |
| `email` | TEXT | NOT NULL | Contact Email |
| `phone` | TEXT | NULLABLE | Phone Number |
| `summary` | TEXT | NULLABLE | Career Objective |
| `updated_at` | INTEGER | NOT NULL | Last edit epoch timestamp |
```
---
```
## 2. Entity: `education_history` (1-to-Many with Profile)
```
| Column Name | SQLite Data Type | Constraints / Keys | Description |
| :--- | :--- | :--- | :--- |
| `edu_id` | INTEGER | PRIMARY KEY AUTOINCREMENT | Primary Key |
| `profile_owner_id` | INTEGER | FOREIGN KEY REFERENCES `resume_profiles(profile_id)` ON DELETE CASCADE | Profile reference ID |
| `institution` | TEXT | NOT NULL | School or University name |
| `degree` | TEXT | NOT NULL | Degree or Certification |
| `grad_year` | TEXT | NOT NULL | Year of graduation |
```
---
```
## 3. Entity: `experience_history` (1-to-Many with Profile)
```
| Column Name | SQLite Data Type | Constraints / Keys | Description |
| :--- | :--- | :--- | :--- |
| `exp_id` | INTEGER | PRIMARY KEY AUTOINCREMENT | Primary Key |
| `profile_owner_id` | INTEGER | FOREIGN KEY REFERENCES `resume_profiles(profile_id)` ON DELETE CASCADE | Profile reference ID |
| `company` | TEXT | NOT NULL | Company name |
| `role` | TEXT | NOT NULL | Job Title |
| `duration` | TEXT | NOT NULL | Dates of employment |
| `description` | TEXT | NOT NULL | Description of duties |
```

## 4. Room DAO Interface
```
```kotlin
package com.saviorsystems.resumemaker.data.database
```
import androidx.room.*
import kotlinx.coroutines.flow.Flow
```
@Dao
interface ResumeDao {
```
    @Query("SELECT * FROM resume_profiles ORDER BY updated_at DESC")
    fun getAllProfilesFlow(): Flow<List<ProfileEntity>>
```
    @Transaction
    @Query("SELECT * FROM resume_profiles WHERE profile_id = :profileId")
    suspend fun getFullResumeData(profileId: Long): FullResumeRelation?
```
    @Insert(onConflict = OnConflictStrategy.REPLACE)
    suspend fun insertProfile(profile: ProfileEntity): Long
```
    @Insert(onConflict = OnConflictStrategy.REPLACE)
    suspend fun insertEducationList(edu: List<EducationEntity>)
```
    @Insert(onConflict = OnConflictStrategy.REPLACE)
    suspend fun insertExperienceList(exp: List<ExperienceEntity>)
```
    @Delete
    suspend fun deleteProfile(profile: ProfileEntity)
}
```
// Relation class to map 1-to-many associations in single query
data class FullResumeRelation(
    @Embedded val profile: ProfileEntity,
    @Relation(parentColumn = "profile_id", entityColumn = "profile_owner_id")
    val education: List<EducationEntity>,
    @Relation(parentColumn = "profile_id", entityColumn = "profile_owner_id")
    val experience: List<ExperienceEntity>
)
