# 09. Testing Assurance Plan

## 1. Developer Checklist Before Release

### Flow 1: Relational Database Integrity
- [ ] Create a resume, add 3 Educations and 3 Experiences.
- [ ] Delete 1 Education. Verify the other 2 remain.
- [ ] Delete the entire Resume. Verify that the cascading delete wipes the orphaned Educations/Experiences from the DB.

### Flow 2: PDF Export Engine (The Hardest Part)
- [ ] Generate a PDF with extremely long text in the "Experience Description" field. 
- [ ] **Verify:** Does the text wrap correctly? Does it push the layout to Page 2 automatically? (Pagination is notoriously tricky in Android Canvas).
- [ ] Export PDF -> Open with Google Drive PDF Viewer. Verify fonts are embedded correctly and readable.

### Flow 3: Storage Access Framework
- [ ] Export PDF on an Android 13+ device.
- [ ] Verify that the file successfully saves to the `Downloads` folder without asking for legacy storage permissions.
