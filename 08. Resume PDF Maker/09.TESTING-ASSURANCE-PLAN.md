# 09. Testing & Quality Assurance Plan

## 1. Automated Test Cases

### A. Unit Tests: `PageBudgetingEngine` (Layout Boundaries)
* **Goal:** Verify that sections are correctly partitioned into multiple pages when their heights exceed the usable print canvas height.
* **Test Case 1:** `singlePageSectionFits`
  * **Input:** 2 sections measuring a combined height of `400f` points. Usable height is `734f` (A4 `842f` - `54f` margins).
  * **Expected:** Returns 1 `PageLayout` containing both sections.
* **Test Case 2:** `overflowTriggersPageBreak`
  * **Input:** 3 non-splittable sections measuring `300f` points each.
  * **Expected:** Returns 2 `PageLayout` pages (Page 1 holds 2 sections; Page 2 holds the 3rd section).
* **Test Case 3:** `splittableSectionDividesBulletPoints`
  * **Input:** An experience block with 10 bullet items (combined height `450f`). Remaining space on Page 1 is `200f`.
  * **Expected:** Splits the block into `part1` (containing 4 bullets that fit in `200f`) and `part2` (containing the remaining 6 bullets), distributing them across Page 1 and Page 2.

### B. Instrumentation Tests: Room Cascading Deletes
* **Goal:** Verify parent-child database relations.
* **Test Case 1:** `deleteResumeWipesOrphans`
  * **Action:** Insert a `ResumeEntity`, add 3 child `EducationEntity` records, and then call `resumeDao.deleteResume()`.
  * **Expected:** Verification query returns 0 records for both parent `resumes` and child `educations` tables.

## 2. Manual Developer QA Checklist

### A. PDF Selectable Text & ATS Validation
* [ ] **Step 1:** Generate a PDF with at least 2 pages of experiences and skills.
* [ ] **Step 2:** Open the PDF in the standard system viewer, Google Chrome, or Canva.
* [ ] **Step 3:** Perform a long-press on any word.
* [ ] **Verify:** Is the text fully selectable? Can you copy-paste the sentences? Use a search filter (`Ctrl + F`) to find a skill. If the text is unselectable, the ATS PDF generator is failing and needs correction.

### B. Local JSON Backup & Import Integrity
* [ ] **Step 1:** Complete a resume profile and tap "Export Backup (JSON)". Save the file.
* [ ] **Step 2:** Navigate back to the Dashboard and delete the resume profile entirely.
* [ ] **Step 3:** Tap "Import Backup", select the JSON file, and confirm.
* [ ] **Verify:** Does the resume card reappear on the Dashboard with all experience bullet points, certifications, and template preferences restored?

### C. Share & Open chooser Intents
* [ ] **Step 1:** Tap "Export PDF".
* [ ] **Step 2:** Verify that on save completion, the app presents the native Android system chooser.
* [ ] **Verify:** Selecting **Canva** loads the vector A4 layout correctly. Selecting a PDF viewer opens the document without rendering errors.

### D. Screen Margin & Scaling Stress Tests
* [ ] **Step 1:** Load the Print Preview screen.
* [ ] **Step 2:** Cycle margins between **Compact**, **Normal**, and **Wide**.
* [ ] **Verify:** The Canvas layout updates instantly, shifting text blocks and updating page numbers correctly without visual clipping.
