# 08. Play Console Policy & Safety

## 1. Permissions Specification
To remain compliance-safe and avoid advanced reviews:
*   **Android 13+ (API 33)**: Request POST_NOTIFICATIONS runtime permission on first launch so the alarm alarm triggers can alert the user when the timer expires.
*   **Exact Alarms**: Use standard background threads or WorkManager alerts instead of SCHEDULE_EXACT_ALARM to prevent policy violations, unless strictly necessary.

## 2. Data Safety Declarations
Inside Play Console data declarations:
*   **Data Collected**: Device IDs (Collected automatically by AdMob Mobile Ads SDK for diagnostic and marketing logs).
*   **Encrypted in Transit**: Yes.
*   **Data Deletion Option**: Users can request deletion (managed via AdMob privacy console).
