# 02. UI/UX Design System & Theme

To ensure distinct visual branding under Play Store spam audits, FocusPulse Timer uses a custom primary layout theme:

## 1. Color Palette Definitions
*   **Primary (Tomato/Red)**: 0xFFE74C3C (Represents focus, intensity, energy)
*   **Secondary (Navy/Slate)**: 0xFF2C3E50 (Represents stability, calm, deep work)
*   **Accent Color**: 0xFFECF0F1 (Light gray background canvas)
*   **Dark Theme Background**: 0xFF1A252F

## 2. Custom Typography (Outfit)
*   Display Large: 56sp, Bold (For timer countdown digits).
*   Title Medium: 18sp, SemiBold (For session labels: "Focus Session", "Short Break").
*   Body Medium: 14sp, Regular (For statistics logs).

## 3. UI Component Specs
*   **Circular Progress Ring**: Surrounds the countdown text, dynamically filling up as time elapsed decreases.
*   **Floating Control Buttons**: Rounded Material 3 buttons to Pause, Play, and Reset the countdown timer.
