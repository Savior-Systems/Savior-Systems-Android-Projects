# 12. Logging & Analytics

All analytics events use the Firebase Analytics SDK.

## 1. Logging Event Schema
*   	imer_started: Logs when a user initiates a study session.
*   	imer_completed: Logs when the work session completes (time left reaches 0).
    *   Param: duration_minutes (Int).
*   d_shown: Logs when AdMob shows interstitial.
    *   Param: d_type (String).
*   sound_scape_unlocked: Logs when user completes a rewarded ad to unlock audio.
    *   Param: sound_type (String).
