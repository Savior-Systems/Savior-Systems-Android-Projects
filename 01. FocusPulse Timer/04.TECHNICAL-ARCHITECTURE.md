# 05. Database Schema & Storage

### DAO Signatures (SessionDao.kt)
```kotlin
@Dao
interface SessionDao {
    @Insert
    suspend fun insertSession(session: FocusSession)

    @Query("SELECT * FROM focus_session ORDER BY timestamp DESC")
    fun getAllSessions(): Flow<List<FocusSession>>

    @Query("DELETE FROM focus_session")
    suspend fun clearHistory()
}
```
## 2. Preference Stores (Jetpack DataStore)
Saves user-defined settings offline:
*   work_duration: Int (Default: 25)
*   short_break_duration: Int (Default: 5)
*   long_break_duration: Int (Default: 15)
*   sound_locked_time: Long (Timestamp until premium sounds remain unlocked)
