# 09. Testing Assurance & QA Plan

## 1. Manual Edge Case Scripts
*   **Scenario A**: User starts timer, pushes app to background, and locks device.
    *   *Expected*: The background thread keeps ticking or matches the timestamp difference upon return.
*   **Scenario B**: System incoming phone call occurs during session.
    *   *Expected*: Timer pauses or continues to calculate timestamp without crashing the interface.
*   **Scenario C**: User triggers ad without network connection.
    *   *Expected*: Ad fails to show cleanly, and the callback immediately proceeds to trigger the break transition.

## 2. QA Standards
*   [ ] LeakCanary reports zero memory leaks after 5 simulated session complete transitions.
*   [ ] Main screen rendering runs at 60 FPS on lower-tier API 24 devices.
