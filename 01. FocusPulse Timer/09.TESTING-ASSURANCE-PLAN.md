# 09. Testing Assurance & QA Plan — FocusPulse Timer
### Version 2.0 · Growth Edition

---

## 1. Core Timer Edge Cases

| Scenario | Steps | Expected Result | Priority |
|:---------|:------|:----------------|:--------:|
| **A — Background + Lock** | Start timer → push to background → lock device for 10min | Timer on return shows correct remaining time based on timestamp delta. Foreground notification still active. | 🔴 Critical |
| **B — Incoming Call** | Start timer → receive phone call → answer → hang up → return | Timer continues or calculates elapsed time correctly. No crash. Notification still visible. | 🔴 Critical |
| **C — Ad Fails (No Network)** | Disable WiFi + mobile data → complete session | Ad fails silently. Callback fires immediately. App transitions to Break screen without hanging. | 🔴 Critical |
| **D — Complete 4 cycles** | Complete 4 consecutive work sessions without reset | Long Break triggers automatically (15 min). Cycle counter resets to 0 after Long Break. | 🟠 High |
| **E — Settings change during session** | Change work duration in Settings while timer is running | Running session uses original duration. New duration applies to NEXT session only. | 🟠 High |
| **F — App kill + reopen** | Force-stop app → reopen within 5 minutes | Timer shows as paused/idle (not running). Session NOT auto-saved as complete. | 🟠 High |

## 2. 🆕 Focus Card Generation Tests

| Scenario | Steps | Expected Result | Priority |
|:---------|:------|:----------------|:--------:|

## 8. Performance & Memory QA Standards

- [ ] LeakCanary: Zero memory leaks after completing 5 full Pomodoro cycles.
- [ ] Systrace: Main screen renders at ≥ 60 FPS on API 24 emulator (low-end device).
- [ ] `FocusCardGenerator.generate()`: Completes in < 500ms on API 24 emulator.
- [ ] Room migrations: Clean DB migration from schema v1 → v2 with no data loss.
- [ ] Widget update: Glance widget re-renders within 60 seconds of timer state change.
- [ ] Ad loading: Interstitial prefetch completes within 5 seconds on a 4G connection.
- [ ] Foreground service: No ANR reported after 5 minutes of background timer operation.
