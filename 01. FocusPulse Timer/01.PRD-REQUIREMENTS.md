# 01. Product Requirement Document (PRD) — FocusPulse Timer
### Version 2.0 · Growth Edition · Target: 1M+ Downloads in 6 Months

---

## 1. Vision & Core Philosophy

FocusPulse Timer is a **premium, viral-loop-powered** Pomodoro study companion designed to dominate the ADHD focus and deep-work niche on Google Play. Its mission is threefold:

1. **Magnetize users** — Be the most feature-complete free focus timer on the market, targeting underserved ADHD and exam-prep communities.
2. **Retain users** — Use streak psychology, weekly challenges, and a home screen widget to keep users coming back daily.
3. **Spread organically** — Turn every session completion into a shareable moment that drives new installs without spending a single dollar on ads.

The application operates under a strict **offline-first paradigm** with zero mandatory signups, zero permissions beyond notification alerts, and zero paid backend services.

## 3. Core Features & Functional Requirements

### 3.1 Timer Engine & Control Loops
*   **Pomodoro Protocol**:
    *   **Focus Session**: 25 minutes (default, adjustable 5–120 min).
    *   **Short Break**: 5 minutes (default, adjustable 1–30 min).
    *   **Long Break**: 15 minutes (default, adjustable 5–60 min).
    *   **Cycle Interval**: Long Break auto-triggered after 4 consecutive Focus Sessions.
*   **Timer States**: `Idle` → `Active` → `Paused` → `SessionFinished` → `FocusCardReady`
*   **Background Persistence**: Timer uses elapsed-time delta calculation so it survives app backgrounding, screen locks, and incoming calls without drift.
*   **Notification Controls**: Persistent foreground notification during active sessions with Pause/Resume/Stop actions operable from the lock screen.

### 3.2 🆕 Focus Card — Viral Sharing Engine
After every completed Focus Session, the app auto-generates a shareable **Focus Card** bitmap image using Jetpack Compose Canvas API:
*   **Card Content**: "I focused for [X]h [Y]m today 🔥", current streak badge, app name watermark ("Made with FocusPulse"), date.
*   **Card Themes**: 3 templates — Minimal (white), Bold (tomato red), Dark (navy).
    *   Minimal theme: free.
    *   Bold + Dark themes: unlocked by watching one Rewarded Ad (permanent unlock).
*   **Share Flow**: One-tap `Intent.ACTION_SEND` image share → WhatsApp, Instagram Stories, Twitter/X, any app.
*   **Zero permissions required**: Uses `FileProvider` + `application/image` MIME type. No `WRITE_EXTERNAL_STORAGE` needed on API 24+.

### 3.3 🆕 Home Screen Widget (Jetpack Glance)
Two widget sizes using Jetpack Glance for Android 5.0+:

**2×1 Compact Widget:**
*   Circular progress ring (current session % elapsed).
*   Session type label ("Focus" / "Break").
*   Today's streak counter.
*   Single Start/Pause action button.

**4×1 Wide Widget:**
*   All compact widget content.
*   Today's session progress (e.g., "3 / 4 sessions").
*   Weekly challenge mini-progress bar.

Widget data syncs via `GlanceAppWidgetManager` update call triggered by `TimerViewModel` state changes.

### 3.4 Focus Soundscapes
Background audio loops to mask ambient noise:
*   **Free**: Rain (bundled `.ogg` asset, no network required).
*   **Rewarded-Unlocked (24h)**: White Noise, Forest, Cafe Ambience, Fireplace.
*   **Unlock mechanism**: Rewarded Ad → 24-hour unlock timestamp stored in DataStore.
*   4 premium soundscapes instead of 2 creates more rewarded ad engagement opportunities.

### 3.5 Session History & Statistics
*   Chronological Room DB log of every completed Focus Session: timestamp, duration, session type.
*   **Daily Summary Card**: Progress toward today's daily goal (e.g., "3/4 sessions complete — 75%").
*   **Calendar Heatmap**: 30-day visual grid colored by session intensity (dark = many sessions, light = few).
*   **Best Day Insight**: "You focus best on Tuesdays ☀️" — derived from Room DB day-of-week query, no network needed.

### 3.6 Streak & Gamification Engine
*   **Daily Streak**: Increments when ≥1 Focus Session is completed per calendar day. Breaks if a day is missed.
*   **🆕 Streak Shield System**:
    *   One free Streak Shield gifted daily (prevents streak loss for one missed day).
    *   Second shield obtainable via Rewarded Ad (once per 24 hours).
    *   Max shield reserve: 2. Cannot stack beyond 2.
    *   Shield UI: Visible on Timer screen as a small shield icon badge.
*   **Milestone Badges** (stored permanently in Room DB):
    *   🌱 Early Bird — Complete a session before 7:00 AM.
    *   ⚡ Deep Worker — Complete 4 sessions in one day.
    *   🔥 Streak Starter — 3-day streak.
    *   🏆 Streak Master — 7-day streak.
    *   👑 Focus Champion — 30-day streak.
    *   🎯 Weekly Warrior — Complete a Weekly Challenge.
*   **Streak milestone celebrations**: Confetti burst animation on 7-day and 30-day milestones using Compose animation API.

### 3.7 🆕 Weekly Focus Challenge
Every Monday at 00:00 local time, a new challenge is activated (calculated from Room DB — no server needed):
*   **Challenge format**: "This week: Complete [X] Focus Sessions"
    *   Beginner: 10 sessions/week.
    *   Standard: 20 sessions/week (default).
    *   Advanced: 30 sessions/week (unlocked after completing Standard twice).
*   **Reward on completion**: Unlock a special Focus Card template for 7 days OR earn one bonus Streak Shield.
*   **Progress display**: Visible on History screen as a weekly progress ring. Also visible in 4×1 widget.
*   **Reset**: Unused sessions do not carry over. New challenge auto-assigns next tier.

### 3.8 🆕 Smart Goal Adaptation Engine
*   After 7 days of usage, the app analyzes Room DB session data and suggests a personalized daily goal:
    *   If user completes 5+ sessions/day consistently → suggest 6.
    *   If user averages 1–2 sessions → suggest 2 (reduces intimidation, improves D30 retention).
*   Shown as a dismissable suggestion card on the History screen: "Your average is 3 sessions/day. Want to aim for 4?"
*   Adaptation is fully local — no server, no ML model. Pure SQL aggregation.

### 3.9 User Settings
*   Sliders for Focus, Short Break, Long Break durations.
*   Daily goal selector (1–12 sessions).
*   Soundscape selector with unlock states.
*   Weekly Challenge difficulty selector.
*   Theme toggle (Light / Dark / AMOLED Black).
*   "Clear History" command with confirmation dialog.
*   Widget quick-setup shortcut.

## 4. Scope of MVP (Phase 1 — Closed Testing)

Build and release for closed testing with these core features:
*   Standard Pomodoro timer engine with background persistence.
*   Streak system (without shield — shield is Phase 2).
*   Session history with Room DB.
*   Basic Focus Card generation (Minimal template only).
*   AdMob: Banner + Interstitial (180s cap) + App Open Ad.
*   English + Bengali localization.

**Phase 2 additions** (before full production launch):
*   Home Screen Widget (2×1 and 4×1).
*   Focus Card Bold + Dark themes (via rewarded ad).
*   Streak Shield system.
*   4 premium soundscapes.

**Phase 3 additions** (v1.1 update, post-launch):
*   Weekly Focus Challenge.
*   Smart Goal Adaptation.
*   Calendar Heatmap on History screen.
*   4-language expansion (ES, HI, TR, PT).
