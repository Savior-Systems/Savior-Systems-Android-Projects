## 1. App Launcher Icon

| Asset | Spec | Path |
|:------|:-----|:-----|
| Adaptive Icon (API 26+) | `ic_launcher.xml` (vector foreground + color background) | `res/mipmap-anydpi-v26/ic_launcher.xml` |
| Legacy Icon (API < 26) | Rasterized PNG from adaptive foreground | `res/mipmap-{mdpi/hdpi/xhdpi/xxhdpi/xxxhdpi}/ic_launcher.png` |
| Round Icon | Same foreground, circular mask | `res/mipmap-anydpi-v26/ic_launcher_round.xml` |

### Foreground Design
*   **Shape**: A bold circular Pomodoro ring with a dynamic pulse fill at ~70% completion.
*   **Icon element**: A stylized tomato-red ring with a white hourglass or pulse wave at the center.
*   **Style**: Clean, flat — no drop shadows on foreground. Bold strokes, minimal detail (visible at 48dp).
*   **Foreground padding**: 25% safe zone inset for adaptive masking on all launchers.

### Background
*   **Color**: Solid `#E74C3C` (Tomato Red) — not gradient. Solid ensures it renders correctly across all adaptive icon shapes.

### Size Matrix

| Density | Pixels | File |
|:--------|:-------|:-----|
| mdpi | 48×48px | `mipmap-mdpi/ic_launcher.png` |
| hdpi | 72×72px | `mipmap-hdpi/ic_launcher.png` |
| xhdpi | 96×96px | `mipmap-xhdpi/ic_launcher.png` |
| xxhdpi | 144×144px | `mipmap-xxhdpi/ic_launcher.png` |
| xxxhdpi | 192×192px | `mipmap-xxxhdpi/ic_launcher.png` |
| Adaptive foreground | 108×108dp vector | `drawable/ic_launcher_foreground.xml` |
