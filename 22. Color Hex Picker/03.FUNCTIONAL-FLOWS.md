# 03. Functional Flows

## App Launch & Onboarding
1. Splash Screen (App Icon + Brand color).
2. GDPR UMP Consent Dialog (EU only).
3. Permission Request (Only if absolutely necessary).
4. Home Screen.

## Core Loop
1. User opens Color Hex Picker Wheel.
2. User executes the primary utility (Extract colors from images or use a color wheel to build offline palettes.).
3. Interstitial Ad displayed on completion (subject to 180s cooldown).
4. Data persisted locally via Room DB (Saved Palettes).
