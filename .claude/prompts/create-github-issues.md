# Prompt: Generate GitHub Issues

Use this prompt to break implementation plans down into actionable GitHub issues:

```
Act as a Technical Lead.
Parse the `/07-implementation-plan.md` file for App [NAME].
Generate copy-pasteable Markdown template blocks for GitHub Issues representing:
- Issue 1: Database & Model setup
- Issue 2: Compose Layout Views
- Issue 3: AdManager & Consent loop setup
- Issue 4: Release QA checks
Include explicit definition of done, acceptance criteria, and package naming requirements.
```
