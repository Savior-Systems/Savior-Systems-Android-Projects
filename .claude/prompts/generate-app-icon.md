# Prompt: Generate Premium App Icon

Use this prompt to generate a high-quality Android adaptive icon for any app in the portfolio.

---

## Prompt

```
Generate a premium Android app icon for [APP_NAME].

Design Requirements:
1. The icon must use the app's primary color from `02.UI-UX-DESIGN-SYSTEM.md` as the background or dominant element.
2. The foreground should be a single, minimalist, recognizable symbol that represents the app's core function (e.g., a water droplet for Water Tracker, a clock for Timer, a mosque for Prayer Time).
3. Style: Flat Material Design with minimal depth shadow. Clean, modern, and professional.
4. Format: Square with rounded corners suitable for Android adaptive icons.
5. NO TEXT on the icon. Icons should communicate through symbols only.
6. The icon must be visually distinct from all other apps in the portfolio. Cross-reference `context/portfolio-context.md` for existing color assignments.

After generating, save the icon to the app's directory with a descriptive filename (e.g., `water_tracker_icon.png`).
```
