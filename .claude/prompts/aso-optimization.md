# Prompt: ASO Description Optimization

Use this prompt to generate store listing descriptions optimized for conversion and search visibility:

```
Act as an ASO Copywriter.
Given the target country [COUNTRY] and language [LANGUAGE], rewrite the App Title, Short Description, and Full Description for App [NAME].
Use the keyword cluster provided in [06-play-store-aso.md](file:///[PATH]/06-play-store-aso.md).
Ensure:
1. Exact keywords are integrated naturally in the first 150 characters.
2. The style represents high-quality utility values.
3. Length parameters strictly match Play Store requirements (Title < 30 chars, Short Description < 80 chars, Full Description < 4000 chars).
```
